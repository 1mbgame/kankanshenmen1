快递公司拒绝托运Note 7 三星召回计划问题多！

三星 Galaxy Note 7 频频传来电池过热引发爆炸的事故，迫使三星全球召回存有电池隐患的旗舰智能手机。单是美国，要召回、换机的 Galaxy Note 7 就高达一百万部。不过，三星要在越短的时间内，召回越多问题手机的目标，似乎难以顺利实现。

由于设备可能存有爆炸隐患，导致部分快递公司如 FedEx 和 UPS 拒绝接收和托运 Galaxy Note 7，导致召回计划进度缓慢，甚至夭折。


图片: http://www.androidcentral.com


美国不少消费者透过三星官网购买 Galaxy Note 7，用户受促将问题手机免费寄回给三星。其中一名用户披露，三星在回复邮件中提到 “目前，基于 FedEx 和 UPS 快递公司拒绝托运 Galaxy Note 7，换机计划必须暂停。三星将会宣布另一项换机计划，取代之前的换机或退款方案。有关计划原本将在今天对外宣布，但最后的细节还未敲定，预料将在下周开始落实新的换机计划。目前，请关注三星官网，以便获取最新的资讯。”

三星也在电邮中指出，好消息是该公司已经找到愿意接收和托运 Galaxy Note 7 的私人快递公司，不过全新的换机计划何时开始，目前还是未知数。消息指出，三星 Galaxy Note 7 预计将在9月28日，在韩国重新上市。
