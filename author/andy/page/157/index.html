Note 7 全球停产停售 三星呼吁用户全面停用保安全

三星旗舰版手机 Galaxy Note 7 疑因电池过热引发自燃事件没完没了，尽管三星启动全球召回计划，并为用户更换改良版的 Galaxy Note 7，但还是屡传爆炸事故。三星电子“长痛不如短痛” 正式发表声明，向全球的合作伙伴发出紧急通知，暂时停售 Galaxy Note 7，而且禁止用户以旧机更换同款的新机。

三星也在声明中，要求所有 Galaxy Note 7 的用户停止使用这款手机，不管是旧版还是新版的 Galaxy Note 7 用户都要关机，以策安全！用户可以寻求退款或更换其他型号的手机。


图片: Reuters


三星强调，消费者的安全是该公司的优先考量。目前，该公司正和美国消费者产品安全委员会合作，调查 Galaxy Note 7 引发的种种问题。该委员会赞扬三星停售和禁止用户“旧机换同款新机”的举措为明智之举。该委员会也呼吁消费者应该响应三星的号召，马上关机，安全第一！

《韩联社》星期一报道指出，三星已经停止生产 Galaxy Note 7。随着 Galaxy Note 7 为三星带来巨大的负面影响，尤其是名誉上的重挫之后，Galaxy Note 7 会否就此退出市场呢？还是依然有重新亮相的机会？市场都在观望。
