Note7 裤袋爆炸！ 美国男子提告三星讨公道

三星旗舰手机 Galaxy Note 7 电池爆炸的个案越来越多，一波未平一波又起！如今，三星还要为电池爆炸事件在美国吃上第一宗官司！

美国佛罗里达一名男子乔纳丹斯特罗贝尔和一般人一样，习惯性的把手机塞在裤袋里，但手机却在毫无预警下无故爆炸，造成他重度烧伤。这名男子周五正式提告三星，成为美国首位因 Galaxy Note 7 电池爆炸而提高三星的手机用户。

根据《路透社》报道，该名男子是在三星宣布召回美国一百万部 Note 7 后，向三星提出法律诉讼，要求三星赔偿医药费、误工费、精神痛苦等抚慰金。三星至今还未就这起案件作任何回应。


图片: http://phandroid.com


28岁的乔纳丹宣称，Note 7 於9月9日在他的裤袋里爆炸，裤子随即起火燃烧，造成他的右腿严重烧伤。为了避免伤势恶化，他试图把手机取出口袋，结果一并烧伤左手拇指。

乔纳丹的代表律师透露，乔纳丹蒙受深二度烧伤，右大腿烧伤面积约一部手机大小。

Galaxy Note 7 爆炸事故如雪球般越滚越大，三星至今在美国接获92宗投报，其中26宗涉及烧伤，55宗涉及财务损失。
