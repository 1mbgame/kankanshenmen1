大马三星用户看这里！Note 7 用户可退款或换机！

三星电子旗下的旗舰手机 Galaxy Note 7 一再传出爆炸事故，让用户无所适从，忧心忡忡。随着三星宣布全面停产、停售 Note 7，并呼吁用户马上关机后，大马三星电子今天正式发布文告，强调大马的 Note 7 用户可以退款或更换其他型号的手机！


三星指出，如果你是 Galaxy Note 7 的用户或是已经向三星预购 Galaxy Note 7 的消费者，可以要求退款或是更换成 Samsung Galaxy S7edge。三星客户服务人员预料将从10月13日开始，联络在官方网站预购 Galaxy Note 7 或是透过合作伙伴，包括电讯公司预购 Note 7 的用户，以便进一步协调退款和换机事宜。

三星也在文告中敦促 Galaxy Note 7 用户马上关机，以策安全。民众若有疑问，可以致电大马三星客户服务专线 1-800-88-7799。三星也为造成用户的不便而道歉。
