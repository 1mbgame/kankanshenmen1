开卖2周传5宗爆炸事故！三星Galaxy Note 7 暂停发货


图片来源：http://m.ppomppu.co.kr/


三星电子8月19日才在韩国推出旗舰智慧型手机Galaxy Note 7，但没想到“出师不利”！短短两个星期内，全球共发生了5宗 Galaxy Note 7在充电时爆炸的事故！

韩国三星电子8月31日宣布，将在Galaxy Note 7生产过程中加入额外的质量测试，Galaxy Note 7为此将延迟发货。不过，三星并没有说明延迟发货是否和电池爆炸事故有关，也没有透露检测内容。目前，韩国三大营运商— SK电信、韩国电信和LG U+已经紧急刹车，暂停发货。

Samsung Galaxy Note 7开卖以来，受到消费者高度关注，单是在韩国的预购量，已经突破40万部，比起3月份面市的Galaxy S7，销量翻倍。



根据韩国媒体报道，多名拿到Galaxy Note 7 的消费者遇上质量问题，手机电池在充电时发生爆炸。三星已经介入调查，暂时还未公布调查结果。不过，从网络上曝光的资料来看，不排除手机电池爆炸和使用第三方充电器有关。但若爆炸事故与手机质量有关，将对三星旗舰产品带来一记沉重打击，还未正式与苹果即将推出的iPhone 7 较量前，已经“输在起跑点”。
