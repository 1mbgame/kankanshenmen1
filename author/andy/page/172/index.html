空中惊魂！三星手机航班上起火冒烟！别以为是 Note 7……

三星最近因为新一代旗舰手机 Galaxy Note 7 电池爆炸事故而忙得焦头烂额，现在破屋惨遭连夜雨，遭遇“空中惊魂”！不过，这回和 Galaxy Note 7 毫无关系，出事的是旧型手机 Note 2……


图片:  https://daniellelatta.wordpress.com

根据路透社报道，印度航空公司 IndiGo 发表声明，证实一趟从新加坡飞往印度金奈，编号为 6E-054 的航班，发生 Note 2 起火冒烟事故。乘客在飞行途中，发现头上的行李格突然冒烟而马上召来空姐。空姐打开行李格后，发现一部三星 Note 2 闪着火花，冒出青烟。

空姐马上用灭火器喷浇手机，然后浸在水桶里，放在卫生间，一直到飞机降落才移交给相关单位。 庆幸的是，这起事件并没有对飞行安全构成威胁，没有人在事故中受伤，班机最终安全降落金奈。

图片: Reuters

三星公司回应说，顾客安全是该公司最优先的考量，公司已经和当局接洽联系，了解事发经过，并介入调查。 三星宣布全球召回 Galaxy Note 7后，这部旗舰版手机就成了人人喊禁的“过街老鼠”，不过，出事的 Note 2 并不在印度民航局禁用的范围内。
