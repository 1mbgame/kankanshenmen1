三星推升级系统防爆炸！限制充电60%

三星 Galaxy Note 7 电池引发爆炸事故接二连三，三星紧急启动全球召回手机计划，不过召回速度太慢，当务之急是采取新措施防止 Galaxy Note 7 再爆炸！

为了防止爆炸事故，三星将会推送新的升级系统，并限制 Galaxy Note 7 的用户每次充电只能充 60% 电量，确保电池密度维持在安全水平，避免电池短路引发爆炸。


图片: http://www.trustedreviews.com



图片: http://gizmodo.com

根据了解，更新版系统是强制性的，Galaxy Note 7 用户必须安装，直到用户更换新的 Galaxy Note 7 后，系统才会解除。新系统预计在 9月20日 推送，同时也会提醒用户，召回 Galaxy Note 7 以换取新机。不过，暂时不确定，除了韩国用户之外，三星是否会针对其他市场推出更新系统。

三星从系统层面限制充电量，是为了满足坚持使用存有风险，但受到召回计划影响的 Galaxy Note 7 用户，保障他们的安全。基本上，限制充电量更新系统只针对召回范围内的用户，新发货的 Galaxy Note 7 用户则不受影响。
