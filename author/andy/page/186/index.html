三星Galaxy Note 7 又“爆”红 ！这次竟“烧车又烧房”！

当苹果风光推出iPhone 7 时，准备以Galaxy Note 7与苹果较量的老对手三星却“栽在自己手里”，再传电池爆炸事故！更夸张的是，这次遭殃的不只是一部手机，而是一辆车，一栋房！

根据美国媒体报道，第一起爆炸事故发生在南卡罗莱纳州。一名男子把新购的 Galaxy Note 7 留在屋内的墙上插座充电，接着出门去接孩子。但没想到，不到一个小时，当他回到家时赫然发现，消拯人员忙着灭火，车库早已一片火海！

这起爆炸事故的罪魁祸首，是不是Galaxy Note 7 还有待调查。因为在同样的插座上，还插有一个没在运作的空气压缩机。


图片来源: https://9to5google.com

第二宗案件则发生在圣彼得堡，一名男子把Galaxy Note 7 留在吉普车里充电，结果同样让人“难以置信”！所幸他的家人，包括8岁的女儿在几分钟前，已经下车回到屋内，否则后果不堪设想。

这名用户稍后再面子书更新状态时写到：“我不愿相信这是Note 7引起的悲剧，但所有迹象都指向三星 Galaxy Note 7。三星已经联络我，之后再也没有回电，我想是时候聘请律师了！”

图片来源: http://www.fox13news.com/


随着全球共传出约35宗因 Galaxy Note 7电池而引起的爆炸事故后，三星已经紧急启动召回计划，并承诺为所有购买Galaxy Note 7 的用户提供换新服务。在苹果强势推出iPhone 7时，被视为三星旗舰产品的Galaxy Note 却灾情不断，叫三星情何以堪？
