换了电池还是“爆”不停 三星停止制造Note 7！

三星强势推出的旗舰手机 Galaxy Note 7 问题一箩筐！就算召回旧机换新机，也不见得改善手机电池过热燃烧起火的霉运！随着改良版 Galaxy Note 7 依然传出数宗爆炸事故后，三星电子公司宣布，暂时停止生产 Galaxy Note 7！

根据《华尔街日报》引述业界人士消息指出，三星宣布停产 Galaxy Note 7，以便专注于全球召回 250万部“问题手机”。

而在较早前，美国两大营运商 T-Mobile宣布，全面停售Galaxy Note 7。T-Mobile还给予送回Galaxy Note 7的用户25美元或约100令吉的通话费补偿。

除了 T-Mobile，美国第二大无线运营商 AT&T也在周末宣布，不再销售或为用户更换新的Galaxy Note 7。T-Mobile 强调，该公司鼓励召回“问题手机”，不过用户可以更换成三星线下的其他智能手机或其他选择。

随着 Galaxy Note 7 停售又停产后，Galaxy Note 7 该何去何从啊？会不会就此退出手机市场的舞台呢？
