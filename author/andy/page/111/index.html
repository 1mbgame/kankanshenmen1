Note 7 为何爆炸？三星召开记者会公布调查结果！

三星一代机皇 Galaxy Note 7 自去年8月19日在全球发布以来出师不利，频传自燃爆炸事故。尽管三星之前曾启动更换手机电池计划，无奈换了新电池后，手机自燃隐患仍存在，最后迫使一代机皇走上停售、停产之路，成为寿命最短的三星旗舰手机。

经过数个月的研究调查后，三星电子在我国时间今早9点于韩国首尔举行新闻发布会，公布 Galaxy Note 7 爆炸原因！三星移动部门主管高东镇在记者会上指出，三星从各源头全方位调查后证实，电池果然是罪魁祸首！

图片: 美联社

高东镇透露，该公司第一批召回的电池和第二批召回的电池不同，但两批电池都有瑕疵！首批手机搭载的是 SDI 生产的电池，因为手机设计采用双曲面设计，内部采用卷绕式电池右上角有瑕疵，出现挤压受损的现象，也等于说，首批电池自燃主要是因为负极板受到挤压所致才会起火。第二批电池则因为正极焊接有毛刺，刺穿电池导致短路起火。另外，还有一部分电池被发现，没有使用绝缘胶带、电池上尖锐突起，也有一些电池的隔离膜较薄，这些原因都是导致 Galaxy Note 7 自燃的原因。总结而言，是电池设计和制造问题，导致 Galaxy Note 7 自燃起火。

图片摘自网络

除了电池外，三星电子也从硬件、软件、物流、快充电、防水、虹膜识别技术、电流等各方面进行测试，显示都和事故原因无关。高东镇在记者会一开始，先向受事故影响的各单位以及消费者致以最诚挚的歉意，并感谢各方对三星的信任和耐心的等待。他指出，该公司为了 Galaxy Note 7 制定新的电池标准，但并没有在推出市场前就电池设计，和制程所引起的问题进行最终确认，三星难辞其咎，必须负起全责。

图片摘自网络

三星也在新闻发布会中指出，在全球一共出售306万部 Galaxy Note 7，截至今年1月23日，三星已经回收超过96%的 Galaxy Note 7手机。
