MWC评选年度最佳手机！谁是2016年最佳“机皇”？

世界移动通讯大会 (MWC) 主办单位 GSMA 协会按照惯例，评选年度最佳智能手机。随着去年推出市场不久就“出师不利”的 Galaxy Note 7 宣告出局后，跻身5强的智能手机分别有苹果 iPhone 7 Plus、Google Pixel XL、Galaxy S7 edge、华为 P9 和 Moto Z。到底谁才是 2016 年度智能手机之中的“王者”？



今年，GSMA 把最佳智能手机大奖颁给了三星 Galaxy S7 edge！有趣的是，Galaxy S7 edge 也是去年 MWC 评选的最佳智能手机。三星在去年的 MWC 大会上正式发布 Galaxy S7 和 S7 edge，S7 edge 采用双曲面屏设计，而且配置也比 S6 edge 来得更强，不管是 2K 显示屏、IP68 防尘防水、PD 传感器、无线充电等等功能都让人眼前一亮。

GSMA 协会赞扬 Galaxy S7 edge 设计感非常强，美感十足。一般相信，如果 Galaxy Note 7 没有因为爆炸事故而黯然退场，那么 2016 年最佳智能手机就会非它莫属。但 Note 7 悲惨的结局却给 Galaxy S7 edge 表现的机会，三星去年下半年顺势推出更大容量、更多配色的 S7 edge 手机，这才二度获得最佳智能手机的殊荣。
