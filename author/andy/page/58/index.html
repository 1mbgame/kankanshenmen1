华为勇夺最佳拍照手机及最佳穿戴设备！

欧洲影音协会EISA近日公布了2017-2018年度大奖，早前在中国品牌日获网民投选为中国第一品牌的华为就不负众望，凭借华为Watch 2 和 P10一举拿下最佳穿戴设备和最佳拍照手机两项大奖！


图片取自cnbeta.com

此外，华为P8 和 P9也曾经连续两年获得最佳消费者手机大奖，不过这项荣耀今年就归荣耀8 Pro了。另外，获得最佳智能手机的是三星Galaxy S8／S8 plus，而最超值手机大奖则是由NOA Element H10le夺得。

最超值手机大奖：NOA Element H10le。图片取自twimg.com

其他奖项包括：最佳随身听大奖FiiO X5 3rd gen、最佳耳机大奖JBL Everest Elite 750NC、最佳喇叭大奖JBL Boombox、还有最佳入耳式无线耳机大奖Sennheiser MOMENTUM In-Ear Wireless 。
