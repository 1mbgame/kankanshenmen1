Note7 爆炸原因三星也不懂？ 韩国政府介入调查“罪魁祸首”

三星 Galaxy Note 7 手机一再传出爆炸自燃现象，迫使一代旗舰手机“走上绝路”，不但全球停产、停售，可能还会面对全部被消灭的窘境，为三星带来数十亿美元的巨大损失。

三星起初认为爆炸事故的罪魁祸首是有问题的锂离子电池，但改良版 Galaxy Note 7 更换新电池后依然爆炸自燃，让三星无计可施，宣布全面停产和销售这款手机。目前，三星电子已经展开内部调查，以鉴定 Note 7 爆炸的原因。


图片: 彭博社

 
根据韩国媒体报道，韩国政府已经委派韩国产业安全技术院 (KTL) 就三星 Note 7 爆炸起火事故启动调查工作，KTL 将对韩国国内发生起火事故的4-5部 Note 7 展开集中调查。 三星电子方面也已经将部分回收的肇事手机送往有关部门，并接受 X光、CT 扫描等先进仪式的检测。当局对 Note 7 进行严密检查，务必找出重挫韩国电子巨头和韩国经济的“罪魁祸首”。

图片: http://www.digitaltrends.com

图片: http://news.softpedia.com


业界分析认为，爆炸原因对三星至关重要，一旦调查报告出炉，三星也理应将调查结果公诸于众，不应把 Note 7 爆炸门当作孤立事件，而应当作教训，并采取有效措施，避免重蹈覆辙。受到 Note 7 事故影响，三星电子已经下调今年第三季度利润预期，三星指出，全面召回并停售 Note 7，对该公司至少造成 54亿 美元或约 220 亿令吉的损失。
