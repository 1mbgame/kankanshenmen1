香港首宗Note7爆炸 "问题手机"或超出预测

三星 Galaxy Note 7 因电池引发爆炸的事件“没完没了”！香港也宣告“沦陷”，发生首宗 Galaxy Note 7 爆炸事故！根据香港媒体报道，发生爆炸的手机并不在三星召回的范围内，也就是说，有关手机并非采用问题供应商提供的电池。这不禁让人质疑，电池存有安全隐患的 Galaxy Note 7，可能超出官方预测的数量。

香港一名女子在面子书上载多张严重烧毁的手机照片，引起关注。他宣称，9月2日在香港电器连锁店购买 Galaxy Note 7，没想到，才使用不到三个星期，就宣告报销！整部手机熔烂犹如黑炭。


图片: http://www.appledaily.com.tw

随着国外日前相继传出 Galaxy Note 7 爆炸事故后，他在购买手机的第二天，曾经致电香港三星公司询问可否更换手机，但对方向他保证，他的手机没问题，不需更换。他指出，手机爆炸前，一度发出 “吱吱吱”的声音，她随即放下手机！果然不出所料，手机开始冒烟，边框随之爆开。所幸他当机立断，否则可能烫伤。

三星香港已经介入调查分析。根据官方检测结果，手机过热融化是因为外来热力所致。三星已经联系有关用户，以进一步了解事发经过。
