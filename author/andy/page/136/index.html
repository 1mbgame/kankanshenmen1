从此消失太可惜？三星或在“特定地区”重新出售 Note 7

三星 Galaxy Note 7 爆发自燃爆炸问题，让韩国企业紧急召回全球 Note 7 手机，同时宣布停产、停售，蒙受史无前例的损失。虽然三星电子和韩国相关部门已经展开调查，但至今原因不明，有关单位仍未对外解释，电池是否是引发手机自燃的“罪魁祸首”。

Galaxy Note 7 作为三星一度引以为傲的旗舰产品，要它彻底消失，就像从来没有出现过一样，似乎太可惜了。最新消息指出，三星似乎也不想彻底放弃 Note 7！


图片: www.thenextweb.com


韩国 引述不愿具名的业界人士谈话宣称，三星可能考虑明年重新出售 Galaxy Note 7 翻新手机，有可能在印度、越南等新兴市场发售。虽然三星还未做出最终决定，但明年重启 Galaxy Note 7 的销售计划，可能性蛮高。

三星这么做，无疑是想把损失减到最低，不过随着 Note 7 自燃原因至今不明，消费者是否会再度考虑 Note 7？恐怕重启销售之路不太容易。
