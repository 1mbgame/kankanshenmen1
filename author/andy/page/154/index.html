三星在机场紧急设柜台 供用户紧急更换Note7手机

三星旗舰手机 Galaxy Note 7 因为爆炸自燃事故而永久性停产，三星早已启动全球召回换机计划，不过还有人未接获召回讯息，也有人不愿放弃 Note 7，坚持不退回、更换手机。随着全球越来越多航空公司禁止被视为“危险物品”的 Galaxy Note 7 登机后，三星采取紧急措施，在全球各大机场设立柜台，供用户退换 Galaxy Note 7。

如果用户不愿当场更换其他型号的手机，三星也让用户快速领取备用机上机，工作人员会现场协助用户简单的备份资料，解决换机后的困扰。消费者抵达目的地后，可以透过三星柜台或官方管道协助相关的后续事宜。


图片: https://www.buzzfeed.com


目前，三星已经在韩国、美国、澳洲、香港、台湾、新加坡等国家的各大机场设立柜台，让携带 Note 7 登机的用户，可以直接在柜台更换其他型号的三星手机，避免乘客的 Galaxy Note 7 在登机前被拦下而面对不必要的损失。三星采取的紧急人性化举措，备受用户和外国媒体的肯定，称三星主动为用户提供便利，又能够尽快回收更多“问题手机”。

目前，禁止乘客携带 Galaxy Note 7 登机的航空公司包括中国春秋航空、马来西亚亚洲航空、日本全日空、新加坡航空、澳大利亚航空、阿联酋航空、维珍航空、新西兰航空、意大利航空、柏林航空和芬兰航空等等。
