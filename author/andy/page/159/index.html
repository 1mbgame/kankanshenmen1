改良版 Note 7 首爆！烧透机上地毯！乘客紧急疏散

三星电子公司因为 Galaxy Note 7 电池过热引发爆炸事故而忙得焦头烂额，全球召回、换机计划已经让三星应接不暇。原以为改良版的 Note 7 可以终结爆炸事故，为三星挽回一些面子和消费者的信任，但没想到，改良版的手机还是“辜负众望”，并且“火上添油！”


图片: http://www.theverge.com

根据知名科技网站 The Verge 的报道，一部改良后的 Galaxy Note 7 手机在美国西南航空公司的一架客机上起火冒烟，乘客和机组人员紧急疏散，迫使有关航班取消。所幸事故没有造成任何人员受伤。美国消费品安全委员会（CPSC）和美国联邦航空管理局（FAA）已经介入调查。

图片: http://www.theverge.com

起火冒烟的机主是布莱恩格林（Brian Green）。他两个星期前收到三星电子的短信，并遵循指示更换改良版 Galaxy Note 7。尽管上机后遵照空服员的要求关闭手机，并将手机放在口袋里，但没想到，手机不小心掉落在地板上时，却开始冒出浓烟。当时，手机还有80%的电量。他赶紧掏出手机扔在地板上。结果，Note 7 烧穿了机上的地毯。

三星电子也发表声明强调，该公司已经追回出事的手机，以鉴定起因。
