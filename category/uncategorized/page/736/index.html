三星也办促销活动！机皇 Galaxy S7/ S7 edge 有折扣

苹果日前举行促销活动，iPhone 7 和 iPhone 7 Plus 一律获得 RM500 折扣，让等了许久舍不得买 iPhone 的朋友能够以“可接受”的价格购买心头好。继苹果之后，马来西亚三星今早11点，也在官方脸书发布帖文，宣布 Galaxy S7 和 S7 edge 将获得 RM500 折扣！这是不是说明，Galaxy S8 很快就要来马了呢？



三星 Galaxy S7 采用 5.1 英寸 Super AMOLED 显示屏 (分辨率 2560 × 1440 像素)，搭载骁龙 820 处理器，配备 4GB RAM + 32GB ROM，摄像头采用双核全像素传感技术和f/1.7 大光圈，还加入 “Always On” 技术屏幕，可在锁屏的情况下实时显示时间，日历等信息。


三星 Galaxy S7 edge 正面采用 5.5 英寸 Super AMOLED 显示屏 (分辨率为 2560 x 1440 像素) ，正面的双曲面屏结合 2.5D 弧面设计，加上背部玻璃面板也采用 2.5D 弧面设计，整体结合带来不错的手感。Galaxy S7 edge 内置骁龙 820/Exynos 8890 处理器，配置 4GB RAM，内置 3600mAh 容量电池，支持快充技术（QC2.0），后置 1200 万像素摄像头，包含 f/1.7 大光圈，同时前置 500 万像素镜头。 

值得一提的是，Galaxy S7 edge 本月初被世界移动通讯大会 (MWC) 评选为“年度最佳智能手机”，横扫 iPhone 7 Plus、Google Pixel XL、华为 P9 和 Moto Z 二度获此殊荣。主办单位赞扬 Galaxy S7 edge 设计感强，美感十足。

Galaxy S7 折扣前的零售价为 RM2,699，Galaxy S7 edge 的售价为 RM3,099。如果你对 Galaxy S7 或 S7 edge 有兴趣，千万不要错过这次的促销活动咯！
