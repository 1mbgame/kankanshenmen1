Note 7 换电池后“卷土重来”？三星官方如是回应

三星 Galaxy Note 7  自燃爆炸事故让消费者闻之色变，全球航班也因此“封杀”这款危险物品。不过，根据韩国媒体《Hankyung》报道，三星有意为 Note 7 换上更小容量的电池后，重新上架销售翻新版的 Note 7，不排除在今年5月或6月在印度、越南等新兴市场“重新出发”，借此减少因 Galaxy Note 7 回收事故带来的巨大损失。

图片: http://3c.ltn.com.tw

如果翻新版的 Galaxy Note 7 重归市场，你会给它第二次机会吗？外国媒体《Gadgets 360》获得三星印度发言人的回应，指三星计划在印度重新出售翻新版 Galaxy Note 7 的报道并不属实！简短的访谈直截了当，但发言人可能仅就印度市场做出回应。三星会否在印度以外的新兴市场重新出售 Note 7 就不得而知了。

图片摘自网络

韩国媒体数天前报道，三星计划更新 250万部 Galaxy Note 7 手机，并将原为 3500mAh 容量的电池换为 3000mAh 的电池，同时采用全新外壳。分析认为，三星推出更新版 Note 7，不但可以避免销毁手机所带来的环境污染问题，还可挽回部分经济损失，重新出售 250万翻新版的 Galaxy Note 7 后，料三星可挽回 52亿美元的损失。
