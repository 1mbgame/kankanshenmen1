配华为Mate 10同款AI处理器 Honor V10售价更低

荣耀V9今年初才问世，荣耀V10也提前报到！作为华为子公司，荣耀旗下的Honor V10是继Huawei Mate 10系列后，第二款搭载麒麟970人工智能(AI)芯片的手机。

荣耀V10跟上潮流，采用5.99英寸全面屏设计。官方指出，荣耀V10突破全面屏手机的设计障碍，虽然内置3750mAh大电池，但全金属一体机身宽度仅为74.98mm，而厚度只有6.97mm。

值得一提的是，与其他后置指纹识别设计不同，荣耀V10采用前置指纹设计。除了这点之外，荣耀V10的另一特点便是搭载与Huawei Mate 10系列同款的Kirin 970 AI处理器，搭载专用处理单元NPU，使得手机性能提升25倍，功耗仅为原来的50分之一。另外，荣耀V10也支援抬手亮屏与脸部识别功能，官方直指脸部识别精度与速度媲美苹果与iPhone X。


除此之外，拍照也是荣耀V10的亮点。荣耀V10后置2000万像素AI变焦双摄，配备双ISP，双摄支援f/1.8大光圈，能够识别13种场景，包括人像、动物、食物、夜景等，并能实时调整拍摄参数，同时具备人像模式、背景虚化、智能运动等功能。

游戏方面，得益于AI学习，荣耀V10对《王者荣耀》做了专项优化，支持高帧率模式，也设置游戏免打扰、锁定导航键、屏蔽信息通知、来电一键免提等功能。

配色提供幻夜黑、极光蓝、沙滩金与魅力红四种配色，4GB+64GB版本售价2699人民币（约1679令吉），6GB + 64GB版本要价2999人民币（约1869令吉），6GB + 128GB版本定价3499人民币（约2179令吉）。 

Honor V10 规格参数：
  • 5.99英寸1080p全面屏（比例为18:9）
  • Kirin 970 AI处理器
  • 4GB/6GB RAM + 64GB/128GB ROM（可透过记忆卡扩展）
  • 前置1300万像素镜头
  • 后置2000万 + 1600万像素双摄像头
  • 前置指纹识别
  • 电池容量为3750mAh
  • 支援5V/4.5A 快充技术
  • 支援面部识别
  • 运行基于Android 8.0的EMUI 8.0系统
