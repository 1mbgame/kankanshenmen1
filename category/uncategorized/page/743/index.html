那么快就出问题？任天堂Switch玩家烫伤手指

任天堂三月初正式在全球发布玩家期待已久的 Nintendo Switch，在短短两周内就售出 150 万台，成绩亮眼！但没想到，一个月还没到，就有玩家爆出被烫伤的消息！

截图自推特

一名玩家 Sos Sosowski‏ 日前在推特上发布推文，透露自己将一张 SanDisk 厂家出品的 128GB SD 卡植入 Switch 时，主机突然出现过热情况，而让人傻眼的是，SD 卡随即融化！他迅速将 SD 卡拔出来，不过 SD 卡已经融化，无药可救！而他的手指也在拔卡时烫伤了。

SanDisk 得知此事后表明，愿意更换一张全新的 SD 卡给 Sos Sosowski‏。其实，除了这名玩家之外，亚马逊也有另一名用户投诉自己的 SD 卡也因为过热而融化。然而，这问题可能和 Nintendo Switch 无关，因为这名用户是把卡插入 GoPro 相机时出现问题。

截图自推特

这名玩家指出，他在 GoPro 上植入 SanDisk SD 卡时，突然闻到塑料烧焦味，他赶紧把卡拔出来，结果发现 SD 卡很烫，而且已经融化烧焦了。至于为何 SD 卡会无故烧焦，过程中到底发生了什么事，就惟有等官方公布调查结果咯！若真是 SD 卡出现问题，就归还 Nintendo Switch 清白吧！
