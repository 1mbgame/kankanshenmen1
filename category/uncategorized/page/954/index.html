这次真的要说“永别”了！三星将"狠心"对待 Note 7！

这回真的要和三星一代旗舰手机 Galaxy Note 7 说永别了！根据外国媒体 BGR 报道，三星电子已经向 Motherboard 证实，三星计划销毁全球所有的 Note 7！

三星因为 Note 7 频频发生自燃情况而全球紧急召回 Galaxy Note 7，不过三星并不打算修理或是翻新回收的 Galaxy Note 7，而是下狠手，一并销毁所有 Note 7。这也意味着，除了那些忽视三星警告，执意使用 Note 7 的人之外，Note 7 将不会存留于市面上。这也表示，三星不会修复 Note 7，没有人能够买到价格较为廉宜的翻新版或改良版 Note 7。三星这次铁了心销毁一代旗舰手机，就好似它从来不曾出现过一样……


图片: BGR.com


三星透露，该公司将采用特定的流程彻底销毁 Note 7，不过三星拒绝透露具体过程。Motherboard指出，包括三星首次召回的 250万台，将有数以百万计的 Note 7 将会报废，这将为三星带来巨大的损失。

如此大规模的全新智能手机直接销毁，也会造成大量金属等资源的浪费。三星如何处理智能手机里难以分离处理的金属，避免对环境造成大面积破坏，也值得关注。

图片: MacDailyNews


Galaxy Note 7 一再发生起火事故，三星已经在全球召回首批 Note 7手机，并以新的电池取而代之，但9月中旬推出更换电池后的新机后，依然频传自燃事故，三星不得不全面对回收 Note 7，并宣布停产、停售，也呼吁用户马上关机！大马三星昨天正式发布声明，宣称已经预购或已经购买 Note 7 的用户，可以选择退款或是更换成 Galaxy S7edge。
