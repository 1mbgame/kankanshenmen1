三星Galaxy Note 8发布日曝光 首批备货量惊人

翻新版的三星 Galaxy Note 7 本月7日才在韩国重新上架，这么快就传来 Galaxy Note 8 的消息！根据韩国媒体《The Investor》报道，三星 Galaxy Note 8 将在8月23日发布，并计划在美国纽约举行发布会，隆重其事推出新一代机皇。


图片摘自网络

消息指称，三星决定提前推出 Galaxy Note 8，并从9月开始全球发售新机。由此看来，三星想抢在苹果发布 iPhone 8 前，提早发布 Note 8 抢攻市场。而另一个原因是，Galaxy S8 系列手机销量不如预期，为此提早发布 Note 8 并不会对 S8 系列销量带来太大冲击。

另外，韩国媒体《The Bell》报道，三星 Galaxy Note 8 首月的备货量为 900 万部，而接下来的三个月，则每月量产 300 万部，也就是说三星计划在短短四个月内出货 1200 万部 Galaxy Note 8。《韩国先驱报》日前报道，3月29日推出的 Galaxy S8 系列至今在全球仅售出 980万部, 但 Galaxy Note 8 首月的备货量已达 900 万部，看来三星来势汹汹，信心满满。

综合此前曝光的消息，Galaxy Note 8 可能搭载6.3寸18.5:9全视曲面屏、双摄和后指纹，预计将预装英文版三星人工智能助手 Bixby，配备支持翻译功能的升级版三星 S-Pen 触控笔等等。随着 Galaxy Note 7 去年因电池过热引发爆炸而全球召回，加上 Galaxy S8 销量平平，三星只能把赌注压在 Galaxy Note 8 身上了，Note 7 的下一代接班人能否拉台销量，惟有等时间和数据告诉我们。
