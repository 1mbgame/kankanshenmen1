6GB RAM + 骁龙821 小米首款双曲面屏手机来马！

大马小米在官方脸书宣布，小米旗下首款双曲面屏幕手机小米 Note 2 正式来马，从4月29日起全面发售。值得一提的是，引进大马的是 6GB RAM + 128GB ROM 版本，售价为 2,499 令吉，仅有黑色一种配色。

小米 Note 2 规格参数
  • 5.7英寸 1080p OLED曲面屏幕
  • Snapdragon 821 处理器
  • 6GB RAM + 128GB ROM
  • 2256万像素后置摄像头 (Sony IMX318 传感器、PDAF 相位对焦+ 反差式对焦、f/2.0 光圈）
  • 800万像素前置摄像头（Sony IMX268 传感器、智能美颜3.0、f/2.0 光圈）
  • 4070mAh 电池容量
  • 支持QC 3.0快充
  • 全功能NFC
  • 运行MIUI 8 系统
小米 Note  2 从4月29日起在小米商城和特定授权经销商正式开卖，当然也会在星期六于槟城开张的全马首家小米之家全面发售！
