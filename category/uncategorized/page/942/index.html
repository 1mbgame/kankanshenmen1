Hello again! 苹果已发邀请函！新一代 MacBook 要来了！

苹果公司每一次的发布会，都会令人瞩目，也是科技界的焦点。较早前，美国科技新闻网站 Recode 引述消息人士指称，苹果将在10月27日，也就是一周后举行新品发布会，推出新款 Mac 系列电脑和笔电。这项消息终于证实了！苹果已经向媒体发出邀请函，确定下周四在库伯提诺总部召开新品发布会。


发布会邀请函打出“hello again”，当年，苹果公司创办人乔布斯也曾在1984年，发布首款 Macintosh 电脑时，在屏幕上打出 “hello”，后来他回归后，发布首款 iMac 电脑，也打出 “hello”，接下来推出第一代 iPhone 时 ，广告词也只有一句台词，也就是 “hello”。

图片摘自网络

由于苹果太空飞船总部将在明年初竣工，因此这一次的发布会将是苹果在总部举行的最后一次发布会。

苹果在年底前推出新升级的 MacBook，在业界的预料之中，基于Mac Book 已经有很长一段时间没有重大升级或突破性的改变，这次的发布会格外让人期待。苹果曾在九月举行的秋季新品发布会上，对外推介新的电脑操作系统macOS Sierra。按照以往惯例， 新款 Mac Book 将会采用新的操作系统，展现新的功能。一般猜测，邀请函上多彩的苹果图样可能预示着苹果即将推出多种配色的 Mac 电脑。

根据早前曝光的消息，苹果计划推出全新的MacBook Pro 笔记型电脑，新产品放弃 MagSafe电源适配器端口和USB-A插口设计，将全部采用USB-C插口，预计新一代的MacBook Pro将采用全新和超薄设计。新电脑的特别之处，可能是谣传多时的OLED功能触控按键，搭载全新的触觉反馈指令系统，还有可能加入和 iPhone 一样的 Touch ID 指纹扫描系统。

报道指出，新MacBook 面市后，苹果可能停售 11英寸MacBook Air。至于12英寸MacBook和13英寸MacBook Air产品的价格会否调低，值得关注。
