Galaxy Note 8 发布在即 三星召回10万部 Note 4！

相信很多人对三星一代机皇 Galaxy Note 7 的既定印象离不开“电池”与“爆炸”，纵使三星官方已经采取多项紧急措施，包括全面回收并停产停售，但电池过热引发爆炸的阴霾仍挥之不去。就在肩负扫除阴霾使命的 Galaxy Note 8 发布在即，三星竟召回 10万部 Galaxy  Note 4！

Galaxy Note 4 已经面市3年，照理而言不会引起大家的关注，但三星这次召回10万部 Note 4，是因为 Note 4 也存有起火风险，罪魁祸首还是电池！美国消费者产品安全委员会宣称，大约10万2千部 Galaxy Note 4 的电池存有起火风险，电池可能会过热而必须召回，用户也受促立刻停止使用需要召回的电池。

不过，这次要对三星公平一点，因为错不在三星，因为这次被召回的“问题手机”是来自美国第二大运营商 AT&T 保修计划中提供的翻新机，由美国联邦快递负责采购。联邦快递与三星已经确认，其中部分是假冒的伪劣产品，并非三星的原厂电池，为此品质备受质疑，存有安全隐患。三星呼吁用户配合美国当局的呼应，更新原厂电池。

基于 Galaxy Note 4 使用的是可更换电池的设计，为此发生在 Galaxy Note 4 的问题不会像 Galaxy Note 7 那般严重，三星只需更换符合规格的电池即可。根据美国消费者安全委员会，该会目前接获一宗手机过热的投报，庆幸的是没有酿成不愉快事件。再过3天，也就是8月23日，三星即将推出 Galaxy Note 8 了，虽然这件事与三星没有直接关系，但现在传出这样的不利消息，还是会对三星带来一丝负面影响吧……
