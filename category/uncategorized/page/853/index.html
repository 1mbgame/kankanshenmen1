继承Note7“未完成使命” Galaxy S8 "非同凡响"

三星 Galaxy Note 7 出师未捷身先死，终成寿命最短的一代机皇，让用户深感可惜，毕竟 Galaxy Note 7 无论在外观、配置、性能或是使用体验，堪称目前安卓智能手机的巅峰之作，也是三星旗下功能最齐全的旗舰手机。集多项功能于一身的机皇走上绝路，估计三星也不会甘心。


图片: http://www.phonearena.com/

根据韩国新闻网站 Naver 报道，三星可能继承 Galaxy Note 7 的“遗产”，将 Galaxy Note 7 的多项卖点、特色和规格，合理移植至 Galaxy S8，其中包括无法在 Galaxy Note 7 上大放异彩的虹膜识别功能。报道指出，Galaxy S8 的指纹识别传感器不再内置于 Home 键之中，而是转移到手机背部，后置摄像头下方。报道也提到，Galaxy S8 不再具备实体 Home 键。

另外，Galaxy Note 7 首次使用的 Y-OCTA 显示屏触控技术也可能移植到 Galaxy S8，这样不但能减少成本、降低显示屏的厚度，使得触控屏显示效果更加亮丽，还能提升触控灵敏度和解决双曲屏误触问题。也有外媒报道指出，三星可能将 Galaxy Note 7 的 Grace UX 接口和 S Notes 迁移至 Galaxy S8，让 Galaxy S8 延续 Galaxy Note 7 未完成的使命。

图片摘自网络

三星因为 Galaxy Note 7 事故而失去太多，需要在智能手机市场上下一番功夫，重建品牌和信誉。无疑，下一代旗舰手机 Galaxy S8 将背负这项重大使命，也使得 Galaxy S8 的表现变得至关重要，成为业界和消费者极其关注的焦点。根据往年惯例，Galaxy S8 预计将在明年2月或3月举行的世界移动通信 (MWC) 展会上正式亮相。
