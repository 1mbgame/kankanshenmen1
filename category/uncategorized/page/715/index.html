晚上睡觉不关Wi-fi 实验结果“很可怕”！你相不相信？

智能手机和网络服务已是我们生活上不可或缺的一份子，可以说是“缺一不可”。因为不想错过太多讯息，不少人晚上睡觉时习惯把手机放在枕边，也不关上 Data 或 Wi-fi。虽然有人质疑 Wi-Fi 会带来对健康不利的辐射，但却没有受到太多人的重视。来自丹麦的5名女生日前就进行了一项震惊科学界的生物学实验！看过这项实验结果后，相信你不会每天开着 Wi-fi 睡觉了！


图片摘自网络

这5名女生和很多人一样，晚上不关手机，而且还要把手机放在床边才能“安心”睡觉，但时间久了，他们发现这么做会导致他们隔天上课时无法集中精神，因此怀疑辐射对人体健康和睡眠品质带来深远的影响。

5个女生为此在设备匮乏的情况下进行实验，他们分别在两个同样室温的房间里放6个托盘，托盘里共有400颗水芹和豌豆种子。他们在第一个房间里设置两台路由器，让 Wi-fi 日以继夜的运行，另一个房间则远离 Wi-fi 信号。

图片: http://stopsmartmeters.org.uk

女孩们每天观察植物的生长情况，给植物浇上同样分量的水，让两个房间里的植物在同等的生活条件下成长。12天过去了，他们得到的实验结果相当惊人！在没有 Wi-fi 信号中成长的种子已经发芽，茁壮成长，但被 Wi-fi 覆盖的种子大多数还未发芽就宣告死亡！

图片: http://stopsmartmeters.org.uk

女孩进行的这项实验让他们获得“年轻科学家”奖项，同时引起科学界的高度关注。来自英国、荷兰和瑞典等国家的科学家对五个女孩的实验结果深感兴趣，纷纷赞扬女孩们巧妙的运用知识进行了优秀的实验。5个女孩“无心插柳”的实验，引起广泛关注，让他们受宠若惊。

图片摘自网络

不过，这项实验引起不少争议。根据《广州日报》早前报道，清华大学工程物理系电磁兼容实验室工程师与北京一家种子专业机构联合进行一次同样的实验，但结果截然不同。两个在模拟环境中的800颗绿豆种子发芽率是一样的！

专家认为，合格的电磁辐射对人体没有明显伤害，并认为 Wi-fi 相较于 2G 或 3G 的手机上网产生的辐射，Wi-Fi 的辐射较小。不管这项实验的结果如何，不管你相信哪一个实验结果，连专家都说 Wi-Fi 的“辐射较小” (也就是有辐射)。所以为了长期的健康着想，还是远离辐射、远离手机好吧！
