小米将发布另一杰作 “双摄旗舰”蓄势待发

最近面市的各品牌旗舰手机都离不开全面屏与双镜头，这两种配置无疑成为今年度旗舰机的“必备规格”。小米官方也发布预告，明示将在9月5日，也就是下个星期二发布一款搭载双镜头的旗舰机！



小米发布的宣传海报打着“Flagship Dual Camera” (双摄旗舰)，预计这部旗舰机将搭载大小光圈或不同像素的摄像头。到底是什么机型呢？有消息称是小米 5X 国际版，但也不排除是传言中的小米 Note 3。

根据早前曝光的消息，小米Note 3将采用双曲面屏设计，配有5.7 英寸AMOLED 屏幕，Snapdragon 835 八核处理器，8GB RAM + 128/256GB ROM，前置指纹识别，后置双摄像头，同时配备 4070mAh 大容量电池。

至于小米 5X，早前已经在中国面市，同样是一款配有双镜头的手机。小米5X采用5.5英寸1080p 显示屏，搭载骁龙 625 处理器，4GB RAM + 64GB ROM ，电池容量为3080mAh，运行最新的 MIUI 9 系统。相机方面，小米 5X 前置500万像素镜头，后置两颗1200 万像素摄像头。

到底小米将在下个星期二发布哪一款手机？是小米 Note 3还是小米 5X？或者两者都不是？大家只能耐心等待 9月5日的到来咯！
