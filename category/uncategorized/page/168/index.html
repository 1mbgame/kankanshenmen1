三星手机电池又出问题 这次轮到Galaxy Note8

每提到手机电池出问题，大家很快就会想起三星一代机皇“Galaxy Note 7”，虽然事隔一年，人们开始淡忘这起事件，但作为Galaxy Note 7继承者的Galaxy Note 8又爆出电池问题，让人想忘掉Galaxy Note 7电池过热引发爆炸事故都难！不过庆幸的是，Galaxy Note 8电池问题的危险程度，并没有Galaxy Note 7那样严重。


图片:https://www.androidauthority.com/

综合外国媒体报道，一些Galaxy Note 8用户纷纷反映，手机电池出了问题，导致手机完全无法使用。不少用户在三星论坛以及国外著名论坛Reddit投诉，指Galaxy Note 8的电量耗至0之后自动关机，不但无法开机，更无法充电！

其中，有用户致电三星客户服务，并根据官方指示，重新设置手机或设法在安全模式下重启手机，但于事无补。一些受影响的用户则尝试更换多条充电线，但还是无法充电。用户表明，手机不曾摔过、也没有进水，一部好好的手机就彻底失效了。

图片:https://gadgets.ndtv.com

外媒Android Authority之后也更新报道，指Galaxy S8 Plus用户也面对同样问题。

根据报道，三星官方建议用户在保修期内归还手机，该公司将依据实际情况为用户更换新机，并强调会立即解决问题。目前，暂时还不清楚Galaxy Note 8电池问题影响范围有多大，但相信三星会尽速解除消费者的忧虑，以免影响机皇的销量。
