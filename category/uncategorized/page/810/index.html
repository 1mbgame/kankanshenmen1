三星已查明Note7事故 爆炸原因率先曝光！

Galaxy Note 7 一度贵为三星重磅出击的一代机皇，却因为频发的自燃爆炸事故让三星忙于收拾残局，一代机皇最后变成“寿命最短的智能手机”，黯然退出市场。经过数个月的研究、调查后，三星将在1月23日于首尔举行新闻发布会，正式公布 Galaxy Note 7 起火爆炸的调查报告。


图片: http://wsj.com/

距离三星发布会还有两天，《华尔街日报》率先公布 Note 7 爆炸原因。该报引述知情人士消息称，部分电池尺寸不合规格，与电池仓不符合，才会导致电池过热，起火爆炸。报道指出，三星将在周一公布这项调查结果，显示 Samsung SDI 提供的电池存在尺寸不一的问题，不合格的电池因为无法适当的嵌入手机内，导致电池与手机尺寸不协调，才会过热。

图片摘自网络

知情人士透露，三星 Galaxy Note 7 的电池分别由 SDI 和中国厂商 ATL 供货。自 SDI 电池引发手机爆炸后，三星将更多电池制造订单转向 ATL。ATL 则因为加大生产量而导致制造过程出现问题，但目前暂不清楚是什么问题。
