峰回路转！微软因这“动人”原因不淘汰Paint了！

伴随微软电脑系统而来的经典画图软体 Microsoft Paint 陪伴了我们32年，但微软决定在推出新版的 Windows 10 Fall Creators Update 后淘汰 MS Paint，并以去年新增的 Paint 3D 取而代之。这项消息传出后，不少粉丝觉得很讶异也很不舍！但没想到，事隔一天，事情就出现峰回路转的局面。


图片: https://blogs.windows.com/

微软在官方部落格中指出，微软看到网友们对 MS Paint 的支持和怀旧实在让人难以置信。“如果这件事让我们学到什么，那就是32年之后，MS Paint 还有那么多支持者。看到那么多粉丝对微软画图应用的热爱，微软意识到经典应用具备惊人的影响力。为此，我们想借此机会厘清一些混淆的地方，并分享一些好消息。”

【MS Paint 将保留下来，它很快将有一个新家 — 在 Windows 应用商店里免费提供。】

图片: https://blogs.windows.com/

太好了！我们不用跟Paint 说再见了！微软也向 Paint 的广大粉丝致谢，谢谢大家对经典画图工具的喜爱。虽然 Paint 还未登陆 Windows 应用商店，而微软也没有宣布软件上架的日期，不过只要得以保留，Paint 什么时候上架，粉丝们都愿意等，相信不用多久就可以等到 MS Paint 啦！
