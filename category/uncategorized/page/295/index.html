红米Note 4今日本地开卖！售价料低于RM900

马来西亚小米这两天在官方脸书陆续释出动态图和预告片后，终于揭露令人振奋的消息！小米宣布，红米 Note 4 将在今天 (3月8日) 中午12点，正式在本地开卖！




小米官方仅公布开卖时间和日期，并没有发布红米 Note 4 的相关配置。到底将在本地发售的红米 Note 4 将和中国版一样，采用联发科 Helio X20 十核处理器（3GB RAM) 还是仿效印度版的骁龙 625 八核处理器 (4GB RAM)？相信引进我国的红米 Note 4 将采用 Snapdragon 625 处理器、配置 5.5 英寸 1080p 屏幕、3GB RAM + 32GB ROM、主相机为 1300万像素 (相位自动对焦)、前置 500万像素摄像头，配置4,100mAh 电池。至于其他详情，则有待小米官方进一步宣布，预计红米 Note 4 售价低于 900 令吉。
