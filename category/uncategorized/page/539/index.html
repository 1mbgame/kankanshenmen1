三星Note 8渲染图曝光！屏幕几乎没有边框

三星下一代旗舰机 Galaxy Note 8 确定将在8月23日亮相，三星已经宣布将于美国纽约举行新品发布会，正式揭开新机的神秘面纱。踏入八月，距离发布日已经越来越近，有关 Galaxy Note 8 的消息、渲染图相继曝光。现在，轮到爆料大神 Evan Blass (@evleaks) 在推特上放出 Galaxy Note 8 最新的渲染图！



这是爆料大神首次爆料号称是 Galaxy Note 8 的相关渲染图，他在推特上发布了这张照片。如无意外，这就是 Galaxy Note 8 的最终设计，手机外形基本上也能确定是长这样了，毕竟爆料大神几乎没有失手过，为此推特上的渲染图可性度极高。


从照片上可以看到，Galaxy Note 8 延续 Galaxy S8 的全面屏设计，边框更窄，屏占比更高，机身看起来更为修长，边角设计也比 Galaxy S8 硬朗一些。可惜的是，爆料大神并没有曝光机身背面。

综合此前曝光的消息，三星 Galaxy Note 8 将配置 6.4 英寸全视曲面屏，首度搭载双摄像头设计，或支持三倍光学变焦，搭载骁龙 835 处理器、6GB RAM，与 Galaxy S8 一样支持虹膜解锁与面部识别，而获得保留的 S Pen 手写笔也将进一步升级。
