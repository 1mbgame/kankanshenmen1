为了让路给Note 8 三星这部新机“被牺牲”

随着三星宣布重磅机型 Galaxy Note 8 将在8月23日发布后，大家对下一代机皇充满想象和期待。综合此前曝光的种种传闻、渲染图和真机谍照来看，Galaxy Note 8 将采取双摄方案已经八九不离十了。当然手机一天未发布，依然存有变数，毕竟三星旗下众多新机型，搭载双摄像头的不止 Galaxy Note 8。

根据爆料大神 onleaks 早前曝光的 Galaxy C10 新机渲染图来看，Galacy C10 延续 Galaxy C9 的经典设计，最大的不同就是 Galaxy C10 搭载双摄像头，是三星旗下首部双摄手机。最近，微博再有人曝光三星 Galaxy C10 手机的真机谍照，照片中的 C10 为黑色机身，双摄像头凸起并竖向排列，双摄右侧则是闪光灯，机身背面搭配微缝天线设计。

图片摘自微博

综合此前消息，三星 Galaxy C10 会采用6英寸 1920 x 1080 显示屏，搭载高通骁龙 660 处理器，6GB RAM + 64GB/ 128GB ROM，采用正面指纹识别，后置双摄像头，电池容量为 4000mAh，支持三星 Bixby 人工智能语音助手。

图片摘自网络

尽管韩国媒体过去曾经披露，Galaxy C10 或在7月首发双摄上市，但三星迟迟没有发放有关 C10 的消息。知情人士爆料，Galaxy C10 已经暂停开发，最主要的原因是让路给8月即将面市的 Galaxy Note 8，让 Galaxy Note 8 拥有更多卖点，首次配备双摄像头将毫无疑问的成为 Galaxy Note 8 的最大宣传噱头。
