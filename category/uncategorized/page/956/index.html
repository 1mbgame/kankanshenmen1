韩国快餐店惊现“火焰汉堡”！员工冒险捡冒烟 Note 7

三星 Galaxy Note 7 “一爆而红”，手机过热自燃事故一宗接一宗！这几天，网络疯传一段视频，乍看之下以为是快餐店员工在捡拾掉在地上的食物，但仔细一看才发现，那是冒烟发烫的手机！


图片翻拍自 YouTube


韩国一家汉堡王快餐店 (Burger King)  星期六发生惊魂事故！一名顾客在仁川汉堡王快餐店里用餐时，放在桌面上的三星 Galaxy Note 7 手机突然冒烟自燃，吓坏了食客！食客马上远离冒烟的手机，不过，一名戴着隔热手套的员工反其道而行，试图把冒烟手机拿走，但手机不慎掉落在地上，期间还听见周围人群的尖叫声！该名员工最后用双手把手机捡起来，只见桌上留下熏黑的痕迹。

其中一名围观者拍下这一幕后，星期六上传到 Youtube，短短数天吸引超过2百万点击率。韩国三星发言人证实，视频中的手机确实是 Galaxy Note 7，但他不愿多说。有关汉堡王餐厅经理随后向媒体透露，店家已经要求三星赔偿餐桌烧焦的损失。

而在美国夏威夷，则有一对夫妇亲眼见证手机在家里冒烟自燃，而监控视频也将整个过程拍摄了下来。事主迪德卡萨指出，“这是新版的 Galaxy Note 7”。她说，事发当时她正在浏览三星网站，想要确认改良版的手机真的没有问题，但没想到握在手里的手机突然开始冒烟！她的丈夫镇定的将冒烟的手机放进铝罐里，等待 911 前来处理。

图片翻拍自 YouTube


随着 Galaxy Note 7 冒烟自燃个案越来越多，三星电子昨天已经正式发布紧急通知，宣布停售、停产 Galaxy Note 7，并要求用户马上关机，不过还是有人不愿交回手机…… 消费者是否应该理智的响应三星的号召，把手机退货，退款或更换其他型号的手机呢？手机只是身外物，别和自己的生命安全开玩笑啊！
