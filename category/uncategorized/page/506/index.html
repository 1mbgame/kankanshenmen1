三星Note 8仅比S8+大0.1寸！新的真的比旧的好吗？

三星机皇 Galaxy Note 7 因为电池过热引发爆炸而出师未捷，粉丝苦等一年终于等到“继任者”。三星终于在23日晚间发布最新旗舰 Galaxy Note 8，为下半年各品牌旗舰手机争夺战打响头炮。有人迫不及待预购，有人说新机与 Galaxy S8 分别不大没有惊喜，也有国外媒体以“Galaxy Note 8 继承 S8 与 Note 7 的精髓” 评价新机皇。Galaxy Note 7 早已离我们而去，Galaxy S8 则在5个月前华丽亮相，如今面市的 Galaxy Note 8 外形上也 Galaxy S8 也没有太大分别，相似度极高。到底两者之间，谁略胜一筹？我们透过参数对比，看看“新的是不是比旧的好”？


图片: http://www.droid-life.com

外形设计而言，Galaxy Note 8 外形设计与 Galaxy S8/S8+ 相似，都采用全面屏设计，乍看之下难以区分，虽然 Galaxy Note 8 换上比 Galaxy S8+ 更大的 6.3 英寸无边框显示屏，比 S8+ 整体尺寸大区区 0.1英寸，不过屏占比更高，边框更窄。若要仔细比较，两者最大的差别就在于机身背面摄像头与补光灯的位置而已，两者相似度极高。

配置方面，Galaxy Note 8 仿效 Galaxy S8 系列，分有 Snapdragon 835 与 Exynos 8895 两种处理器，视不同地区而定。不同的是，Galaxy Note 8 的内存升级到了 6GB RAM，比 Galaxy S8 系列的 4GB RAM 规格高一些，但电池容量比 Galaxy S8+ 小一些。两者同样支援虹膜、指纹及脸部辨识功能、支援 Bixby 人工智能助理、支援 Samsung Pay。简单来说，两部旗舰机在参数配置上未有明显差距。

当双摄已经成为旗舰机的标准配置时，配有两颗镜头的 Galaxy Note 8 自然略胜一筹，毕竟 Note 8 是三星旗下首款配置双镜头的电子设备，也是当今市场上唯一两个镜头都支援 OIS 光学防抖技术的旗舰手机。

Galaxy Note 8 后置1200 万像素双镜头，包括广角与长焦，前置 800 万像素摄像头。值得一提的是，Galaxy Note 8 加入不少拍照新功能，包括拍完照后可以调整模糊程度，还可以在拍照时同步使用二个镜头，做到一次拍摄获得一张广角与一张长焦照片的效果。

除了双摄像头之外，Galaxy Note 8 与 Galaxy S8 最大的差别就是数代 Note 系列的鲜明配件 — 手写笔 S-Pen！三星 Galaxy Note 8 配置升级版S- Pen，不但拥有 0.7mm 极细笔尖、4096 级压感，还支持 IP 68 的防水防尘。S-Pen 也添加其他新功能，包括“即时翻译”，S-Pen 只要放在外语文字上，系统就会即时翻译整行句子，而非像以往那样只是翻译单字。另外，S-Pen 也可以用来做笔记和绘画，增加乐趣。

图片: http://www.businessinsider.my/
售价方面，虽然三星还未公布 Galaxy Note 8 的价格，但可以预计的是，多处升级的 Galaxy Note 8 售价应该会比 Galaxy S8+ 推出市场时来得高，预计将突破4千令吉。严格来说，论参数、配置，设计而言，三星 Galaxy Note 8 确实没什么可挑剔之处，不愧为三星与安卓旗舰，所有创新与功能都以实用与安全为基础，也正因为这样，Galaxy Note 8 少了让人眼前一亮的冲击与惊喜！如果你准备入手安卓旗舰，或许 Galaxy Note 8 是你的选择，但如果你已经拥有 Galaxy S8/ S8+，那就没有必要升级啦，毕竟相似处太多，还是为自己的荷包着想吧！
