三星Galaxy S8 是明年唯一旗舰？ Note 8 命运又如何？

三星电子因为 Galaxy Note 7 爆炸自燃事故而忙得焦头烂额，面临空前巨大亏损。该公司不得不严控产品线，确保新产品的质量，避免公司多年来建立的口碑和品牌“毁于一旦”。

根据 Korea Herald 引述韩国媒体 Newsis 的报道，Note 7 可能是 Galaxy Note 系列中的最后一部手机。尽管三星没有明确说明将永久性砍掉 Galaxy Note 产品线，不过一般分析认为，让 Note 系列产品暂时淡出人们的视线也不失为明智之举，等 Note 7 事故被淡忘后才“东山再起”。


图片: http://www.businessinsider.in


另外，韩媒也报道，三星最近重大更动旗下智能手机的战略，推翻一年推出双旗舰手机的决定。这也表示，三星将在未来一段时间，每年只推出一款旗舰设备。如果三星在2017年只会推出一款高端旗舰手机，那么这款旗舰非 S8 莫属。换言之，三星会集中火力保证 S8 的质量和口碑，同时加大产品测试的力度，确保即将推出的高端机不会重蹈 Note 7 的覆辙，让三星电子名誉扫地。至于 Galaxy Note 8 何时复出？暂时遥遥无期！


根据之前曝光的消息，随着三星宣布生产10纳米制程工艺芯片，并透露将用到明年初问世的产品后，相信 Galaxy S8 将是首发产品。除了搭载骁龙 830 和自家的 Exynos 8895 处理器，还会采用全新的 Mali-G71 GPU。基于三星已在韩国申请注册 LightUp Camera 和 Light+ Camera 两个和相机有关的商标，为此不排除三星的下一代旗舰设备极有可能设有双摄像头。

尽管 Galaxy Note 7 已经全面停产，但三星暂时不打算提前推出 Galaxy S8，预料最快也要等到明年2月底，才能揭开 Galaxy S8  的神秘面纱！
