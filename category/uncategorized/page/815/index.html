还记得 Game Boy 吗？“童年回忆”回来了！

说到经典的手提游戏机，大家会想起共同的回忆 Game Boy 吧！尽管 2005年推出 Game Boy MICRO 后，就被 Nintendo DS 系列取而代之，但 Game Boy 在很多玩家的心目中仍是无可替代的经典。如今，这款伴随着不少人成长的手提游戏机要回来了！虽然不是任天堂旗下的产品…… 

图片来源：Retro Bit Gaming Facebook

游戏开发商 Retro-bit 公司日前在 CES 展会上复刻经典，推出 Retro-bit 版本的 Game Boy—- Super Retro Boy。这款游戏机延续 Game Boy 的外形设计，支援 Game Boy、Game Boy Color 和 Game Boy Advance 的游戏卡带，同时加入了新元素，在原有的 [AB 键]下方增加 [LR键]，以应付多元化游戏。

图片来源：Retro Bit Gaming Facebook
Super Retro Boy 配备 2500mAh 可充电电池，续航力高达10个小时，还配有防刮防碎的 TFT 高清荧幕。Super Retro Boy 售价为 80美元或约 360 令吉，预计将在今年8月正式开卖。
