荣耀调整“双摄手机”价格 Honor 6X折扣高达RM400！

智能手机市场推陈出新，几乎每个月都有新机报到，也因为这样，有些厂商就会举行促销活动或是调整价格，刺激销量！最近，一款堪称“高性价比”的双摄手机就获得官方调整售价，折扣高达400令吉！


没错，说的就是Honor 6X，Honor 6X不但是华为首个采用麒麟655 八核处理器的智能手机，还采用当前最流行的双摄像头设计！随着官方调整售价后，Honor 6X(3GB RAM + 32GB ROM)售价将从原有的1,199令吉调降至899令吉，折扣高达300令吉。而Honor 6X(4GB RAM + 64GB ROM)价格则从原本的1,399令吉减至999令吉，折扣高达400令吉。

Honor 6X 规格参数
  • 5.5英寸2.5D弧面玻璃Full HD显示屏
  • Kirin 655八核处理器
  • Mali-T830 MP2图形处理器 
  • 3GB/ 4GB RAM + 32GB/ 64GB ROM
  • 3340mAh电池容量
  • 支持双卡双待
  • 支持指纹识别
  • 后置1200万+200万像素双镜头 (黑白+彩色)
  • 前置800万像素自拍镜头
  • 售价: RM899 (3GB RAM + 32GB ROM）、RM999 (4GB RAM + 64GB ROM)
