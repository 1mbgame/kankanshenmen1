历代Note中最好战绩 Galaxy Note 8在韩销量火爆！

三星一代机皇Galaxy Note 7去年出师不利而被召回后，许多Note支持者深感失望，期待新一代机皇降临。作为三星最新旗舰，Galaxy Note 8日前强势登场，并自问世以来占据韩国手机销量榜首，预定量突破85万台，比Note 7多出40万部，打出漂亮战绩。


市场调研机构Atlas Research数据显示，Galaxy Note 8发布以来，销量保持稳健增长态势。自10月开始，每日在韩国的销量维持在一万至两万台，单是10月份的第二周，Note 8销量就占韩国智能手机销量的28.3%。

研调机构指出，Galaxy Note 8在历代Note系列手机中获取最好战绩。该机构预计，Galaxy Note 8到年底的出货量，有望突破1000万台！
