红米新机强势来袭 官方：完虐同价位手机

随着红米 Note 5A 日前现身中国工信部后，小米正式宣布旗下新机将在8月21日，也就是明天正式亮相！官方自爆，相较高档的柔光自拍手机，红米 Note 5A 的自拍功能有过之而无不及！而且还有 2+1 卡槽设计，也就是说可以同时使用两张 SIM 卡和一张 TF 卡，容量最高可拓展至 128GB！



工信部曝光的信息显示，红米 Note 5A 采用 5.5 英寸 720p 显示屏，搭载 Snapdragon 425/625 处理器，2GB/3GB/4GB RAM + 16GB/32GB/64GB ROM，电池容量为 3000mAh。

相机方面，小米总裁林斌在微博自爆，红米 Note 5A 配置 1600 万像素前置镜头，同时配备柔光灯和36级美颜功能，自拍效果完爆同等级、同价位的手机，与其他品牌高档的柔光自拍手机相比，有过之无不及。

林斌也在微博上直指红米 Note 5A 是高颜值、高品质、超高性价比的手机。按照惯例，红米 Note 5A 的价位不会超过 1399 元（约 900 令吉），如果一部数百令吉的手机能做到高档次手机的自拍水平，那是厂商的成就也是消费者的福利啊！到底红米 Note 5A 的售价如何，21日自有分晓！官方将在晚间7点30分，以网上直播形式正式发布红米 Note 5A！大家敬请期待~~
