每秒卖出两台！三天狂售50万手机！ “他”做到了！

中国著名手机品牌小米进军印度市场两年来，终于理解到印度节庆有利于手机销量的事实。是的，每年的10月或11月，印度人将欢庆屠妖节或万灯节。小米也看准了印度节日季的商机，在印度举行大促销。

结果，不出所料，小米从10月1日至3日，短短的72个小时，就在亚马逊、Snapdeal 和 Flipkart 三大电商平台上，售出超过50万台手机。这般惊人销量等同于几乎每一秒钟就卖出了两台手机！去年，小米花了一个月才卖出50万部手机啊！


图片: http://blog.xiaomi.com


到底是小米的哪一款手机如此畅销？最新战报显示，红米 3S 荣登 Flipkart BBD 最畅销手机、红米 Note 3 则在亚马逊 Great Indian Festival Sale 销量榜独占鳌头，而小米手环2 则是亚马逊销量最佳的可穿戴产品。

印度有近13亿人口，是仅次于中国的全球第二大人口大国，理所当然的成为全球第二大手机市场，许多手机品牌陆续进军印度市场，分一杯羹。小米今年每个季度都在印度售出 100 万至 150 万部智能手机。如今，随着印度节日季为小米创下惊人销量，相信今年的第四季度，有望成为小米自 2014 年进军印度市场以来，销量最好的一季。
