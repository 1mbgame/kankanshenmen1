Galaxy Note 8 售价太高？三星还有一款中端双摄手机！

三星推出旗下首款搭载双摄像头的新一代机皇 Galaxy Note 8，作为今年度旗舰行列的其中一员，Galaxy Note 8 实现了全面屏与双镜头设计，不管是外形或配置都被视为“无可挑剔”。也因为这样，机皇售价约4千令吉，叫人却步。如果你想购买一部双摄手机，但又没有太多预算，那就耐心等待一下，因为 Galaxy Note 8 不是三星旗下唯一的双摄手机！


根据国外媒体 thaimobilecenter 爆料，三星将推出一款双摄版本的 Galaxy J7。报道指出，搭载双镜头的 Galaxy J7+ 采用一体化金属机身设计，配备5.5英寸全高清AMOLED显示屏， 搭载主频为2.39GHz的八核处理器，4GB RAM + 32GB ROM（可透过micro SD卡扩展至256GB），电池容量为 3000mAh，前置指纹识别，双卡双待，运行Android 7.1.1系统。

文中提到，作为一款双摄手机，Galaxy J7+ 将配备一颗1300万像素的主要摄像头（f/1.7 光圈）以及一颗500万像素的副摄像头（f/1.9 光圈）。虽然目前暂时不清楚 Galaxy J7+ 是否采用与 Galaxy Note 8一样的长焦与广角方案，但据悉能够支援“实时对焦”，实现背景虚化功能。

图片: www.thaimobilecenter.com

值得一提的是，Galaxy J7+ 被爆支援脸部识别，并搭载 Bixby 人工智能助理，有黑色、金色与粉色三种机身配色。Galaxy J7+ 目前还未正式发布，但泰国媒体已经抢先报道，预计 Galaxy J7+ 近期内将在泰国发布。三星 Galaxy J 系列主要面向年轻用户，定位于中端市场，为此预计搭载双镜头的 Galaxy J7+ 售价不会比 Galaxy Note 8 来得高！
