真的“天上掉馅饼”！三星特殊宣传方式让乘客笑不拢嘴

三星一代机皇Galaxy Note 7去年推出后出师不利，至今被全球航空公司禁止带上飞机。事隔一年之后，如今三星采取“在哪里跌倒就在那里站起来”的宣传方式，把全新的Galaxy Note 8带上Galaxy Note 7被禁的机舱内，为机上200名乘客送上免费的Galaxy Note 8！


图片摘自网络

这200名幸运的乘客日前搭乘伊比利亚航空公司航班，从西班牙拉科鲁尼亚飞往马德里巴拉哈斯，他们事前根本不知道三星为他们准备了意外惊喜，乘客们收到礼物时受宠若惊，笑得合不拢嘴！有乘客笑说“收到的礼物比机票还贵呢”！

图片摘自网络

三星在礼物盒子写道“一年前，我们要求你把它关掉，今天我们在机舱内迎接你。”三星以自嘲方式为新机宣传，除了作为回馈用户的一种方式外，也证明三星已经从Galaxy Note 7的阴霾中走出来。三星西班牙更在推特发文 — “三星不仅会在跌倒后重新站起来，我们还从中学会了飞翔！”

可惜的是，这不是全球性的活动啊！大马三星什么时候也和本地航空公司合作，为乘客献上这份意外惊喜呢？
