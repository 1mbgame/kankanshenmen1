还在埋怨iPhone存储空间不够？赶快升级iOS 10.3！

苹果28日凌晨发布更新版操作系统 iOS 10.3，带来新的 Apple ID 页面、App Store 评分机制、Find My AirPods 、强化文件加密和 Apple File System (APFS) 等等新功能。其中备受关注的是 APFS，因为用户发现升级新的 iOS 后，手机的流畅度感觉更高，而且设备的存储空间似乎变多了，这根本是附加“福利”！



是的，这是苹果引入全新文件系统 APFS 的功劳，作为旧文件系统 HFS+ 的替代系统，APFS 具有更好的兼容性。设备的存储空间之所以变大，是因为在 APFS 系统中，就算同样的文件被复制10次、100次，该文件在系统中仅占用一次空间。举个例子，若一个 1GB 的档案被复制5次，在旧的 HFS+ 系统里，档案将拥有5个不同的备份，占用 5GB 空间。但在新的 APFS 系统里，该文件仅占用 1GB 空间。这么一来，设备的空间自然增加了。

根据外媒 Apple Insider 的测试，iPhone 7 Plus 升级到 iOS 10.3 后，设备的存储空间从原本的 75.45GB 增加到 83.26GB，差异为 7.81GB！就算升级后存储空间增加 1至 2GB，对苹果用户而言还是“重大的好消息”啊！
值得一提的是，相较于 HFS+，APFS 采用更高级别的加密标准，具有更高的安全性。除了节省空间，APFS 更加快速、安全，相信更多第三方应用程序会也会随苹果步伐趋向 APFS 功能，为此节省更多空间！所以，还等什么？赶快升级 iOS 10.3 吧！
