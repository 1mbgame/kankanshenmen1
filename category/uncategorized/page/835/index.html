2016 最倒霉公司包括雅虎！最差劲电子产品呢？

2017年来咯！外国各大科技网站纷纷盘点 2016 年“最倒霉”的科技公司和最糟糕的科技产品。若要你点出最差劲的电子产品，“谁”会羞登榜首呢？
根据美国财经媒体 CNNMoney 报道，雅虎和三星电子可谓“流年不利”，双双登上“最倒霉”科技公司排行榜。雅虎不但裁掉 15% 的员工，还因为两度遭黑客攻击而导致10亿用户的信息外泄，引发重大的信息安全问题。雅虎的股价至今已经重挫 14%。


三星电子则因为一代机皇 Galaxy Note 7 自燃爆炸风波而度过最黑暗的一年。尽管三星采取应对措施，以全新电池取而代之但问题非但没有解决，而且涉及层面越来越广，最后迫使三星尴尬收拾残局，不但全球召回问题手机，还宣布停产、停售，让三星遭遇前所未有的财务及名誉损失。三星要如何挽回庞大用户的信心，可是2017年的最大挑战、最优先的任务。

图片摘自网络

而根据美国 Gizmodo 和 Engadget 等科技网站盘点的 2016 年度最糟糕电子产品，则非 Galaxy Note 7 莫属了。Galaxy Note 7 问世时，各大科技媒体都相当看好这款旗舰手机，甚至被视为“市面上最好的安卓手机”，岂料 Galaxy Note 7 不争气，从天堂直坠地狱，成为三星最失败的作品。然而，Galaxy Note 7 仍有大批拥护者，目前韩国仍有约 10万人使用着 Galaxy Note 7。三星惟有使出杀手锏，透过系统更新远程控制，阻止用户继续充电，甚至让手机完全无法再使用。
