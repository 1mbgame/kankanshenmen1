突破技术障碍！全球首部屏下指纹手机一月登场

随着全面屏成为当前智能手机最流行的元素后，业界纷纷把目光锁定于全新的重大技术 — 屏下指纹识别。虽然早前有消息称，三星Galaxy Note 8或苹果十周年纪念版iPhone X将会采用屏下指纹识别，但传言早已不攻自破。最近，技术供应商Synaptics（新思国际）就爆料，在2018年1月举行的CES大会上，将有“大厂商”即将发表全球首部支援屏下指纹识别的智能手机！


图片: https://www.anandtech.com

可惜的是，Synaptics并没有透露到底是哪个“大厂商”，只表示是全球5大智能手机厂商之一，目前已经和对方合作达成共识，而屏下指纹识别传感器也已经投入量产。

图片: Synaptics

Synaptics宣布推出全球首款屏下指纹传感器，型号为Clear ID FS9500。一般而言，如果不是厂商自主研发，多数将采用Synaptics的Clear ID屏下指纹识别技术。根据官方介绍，传感器将直接内置于AMOLED英亩模组中，采用与OLED相同的光学材料和工艺，最终厚度可做到低于1.5mm。

图片: Synaptics

配置屏下指纹识别传感器的手机能提升荧幕屏占比，做到真正的全面屏，为此这项重大技术对全面屏的未来发展扮演重要角色，各厂商都想“能人所不能”。不过，三星之前已经表明，下一代旗舰Galaxy S9并不会采用屏下指纹识别技术。那么，Synaptics所说的“大厂商”会是谁呢？大家只能拭目以待
