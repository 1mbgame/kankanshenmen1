一Click就“中毒”！一张图就能骑劫你的账户！

你都在用 WhatsApp 和 Telegram 吗？如果答案是“是”，那就要特别小心了！因为黑客只要凭着一张照片，就能骑劫你的账号，取得所有短讯、图片和相关讯息，甚至还会用你的名义发送恶意短讯！


以色列电脑保安公司 Check Point 指出，黑客会把含有恶意程式码的照片发送给用户，用户如果点击照片就会中招！黑客会透过照片启动恶意程式码，骑劫用户的账号把病毒散播出去，向用户 WhatsApp 或 Telegram 里的联络人发送恶意讯息，让病毒一传十，十传百。黑客还可以查看并操纵对话，取得受害者个人和群聊记录、查看照片、视频、音频和其他文件。

图片摘自网络

虽然 WhatsApp 和 Telegram 使用点对点加密技术，只有发送者和收件者才能看到讯息内容，但也因为这样，反而令 WhatsApp 和 Telegram 无法分辨讯息是否含有恶意程式码。不过，Check Point 表示，这个安全漏洞只影响 WhatsApp 和 Telegram 电脑网页版用户，估计数以十亿的用户将受影响！

不过，不用担心，WhatsApp 和 Telegram 收到漏洞报告后，已经完成修复工程。但还是要提醒一句，下次收到来历不明的图片时，还是小心为妙啊！
