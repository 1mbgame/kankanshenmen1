Pokémon GO 第二波旋风来袭！80只新宠等你收服，还增新玩法

Pokémon GO 自去年8月正式上线以来风靡全球，但却经不起岁月的考验，即使昨日才结束的情人节特备活动，似乎也激不起玩家的兴致。显然的，若 Pokémon GO 依然故我，很快就会被玩家遗忘。或许开发商也知道玩家耐性有限，为此终于决定一次过释出80多只第二代新精灵！这还不止，游戏还增添新道具、新玩法，相信这将再次点燃玩家的激情，掀起第二波抓宝热潮！

Pokémon GO 在官网宣布，游戏将在本周迎来重大更新，其中将有80多只第二代精灵登场，以及部分第一代可进化成第二代的精灵。


《更新版 Pokémon GO》

新增精灵: 超过80种来自 Johto Region 的新精灵将加入阵容，更多雌雄性别差异的精灵加入行列。

新的进化方式: 一些来自 Kanto region 的精灵很快能进化成 Johto Region 的精灵。简单来说，之前无法进化的精灵，现在解除封印可进化了。玩家需到 PokeStops 领取新的进化专用道具，才能进化相关精灵。

新的玩法: 小精灵将会有新的反击动作，当你捕捉精灵时，别让它们的反应给吓到。另外，在游戏页面上也将新增项目转盘，玩家可立即选择 Berries 或 Poke Balls。

新增两种 Berries: 游戏新增 Nanab Berry 和 Pinap Berry。Nanab 可让精灵的动作放慢，而 Pinap 可让玩家捕获精灵时，获得双倍糖果。

新增服装配件: 玩家可以发挥创意，为训练师更换新的帽子、衣服、裤子和其他配件。

尽管官方仅称 Pokemon GO 第二代“本周”登场，但网传确切日期为 2月18日。这一次，会否再次激起你的热情，归队 Catch’Em All 呢？
