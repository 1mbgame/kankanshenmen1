消息首次曝光！三星 Galaxy Note 8 将卷土重来？

三星电子过去几年秉持“一年双旗舰”策略，上半年由 Galaxy S 系列打响头炮，下半年则由 Galaxy Note 独挑大梁。然而，去年因 Galaxy Note 7 出师不利，导致三星疲于奔命，一方面忙于召回手机，一方面将 Galaxy S7 edge 的宣传和销售工作进行到底，把损失减到最低。也因为 Note 7 事故，三星推迟发布 Galaxy S8，务必确保新一代旗舰机没有任何问题才推出市场。


图片: http://www.knowyourmobile.com

正当大家把焦点放在3月29日即将亮相的 Galaxy S8 时，首度爆出有关 Galaxy Note 8 的消息！是的，三星公布 Galaxy Note 7 爆炸原因时曾经强调，三星不会放弃 Galaxy Note 系列。根据外国媒体 SamMobile 独家报道，Galaxy Note 8 在三星内部的产品代号为【Great】，工程代码定为 SM-N950F，料已经进入开发阶段。如果一切顺利，Galaxy Note 8 将在今年下半年重新归来。

图片: https://www.forbes.com

Galaxy Note 7 去年推出市场时掀起热潮，手机的外形与配置深获好评，而这也是为何三星宣布召回手机时，不少人感到失望，也有不少人不愿意退回手机。一般认为，如果 Galaxy Note 7 没有“遭遇不测”，或许早已荣获英国权威杂志《Mobile Choice》评选为2016年最佳行动装置，也在今年2月举行的世界移动通信大会上获得 MWC 2017 最佳智能手机的殊荣。这一次，三星将 Galaxy Note 8 的产品代号定为【Great】，那么新一代旗舰机到底有多特别、有多【Great】？大家只能期待更多有关 Galaxy Note 8 的消息曝光了。
