macOS High Sierra公测版今日上线 世界上最快桌面浏览器降临！


图片取自AppleInsider

苹果公司三天前宣布公测版新系统上线，但当时也只是发布了iOS 11和tvOS 11的公测版，而macOS High Sierra公测版则在今天才正式提供下载。一般用户只需要在苹果公测版网站注册就可以下载体验了。

macOS High Sierra基本上就是上一版“macOS Sierra”的加强版，主要升级和加强的方面就包括：
  • Safari浏览器的速度提升，苹果甚至宣称它是“世界上最快的桌面浏览器”，JavaScript性能比谷歌Chrome快80%。同时，还会阻止视频广告自动播放，并且加入了防追踪机制，以防用户隐私泄露。
  • 邮件加入spotlight搜寻，支持分屏操作（Split View），磁盘占用空间压缩35%。
  • 采用苹果的APFS（Apple File System）文件系统，效率更高。
  • 支持H.265（HEVC）视频解码硬件加速，同时还有各种编码器。
  • 照片功能提供新的照片编辑器，采用人工智能引擎，让分类更为准确，并开发第三方照片打印功能。
  • 推出Metal 2图形引擎，在图形效能方面比Metal提升高达10倍。MacOS中的Finder也将支持Metal 2，使画面更加流畅和高效。通过雷电三（ThunderBolt）接口还可连接第三方的外接显卡来提升电脑的GPU性能。另外，Metal 2还支持VR功能，同时也支持steam VR。

话虽如此，公测版并不是正式版，只是为了让用户提前体验。如果想要稳定升级的话，还需要等到9月正式版才会到来咯。
