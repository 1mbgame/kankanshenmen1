“没有一部手机比 Note 7 好”！三星铁粉忠诚度惊人！

三星 Galaxy Note 7 陷入“爆炸门”这段期间，恰逢苹果推出了市场瞩目的 iPhone 7 和 iPhone 7 Plus，让三星手机的销量雪上加霜。最近就有一项调查显示，大约 48% 的 Galaxy Note 7 用户，正在考虑转投 iPhone 7 的怀抱！不过，在三星正式宣布停产、停售、停换Note 7 时，还是有三星的超级铁粉对三星不离不弃，甚至罔顾自己的安全，坚持把存有安全隐患的 Galaxy Note 7 留在身边！


图片: Android Central

在三星正式宣布放弃 Galaxy Note 7 后，不少三星铁粉纷纷在社交媒体表达不舍之情！有人说，不会更换其他型号的手机，也不会遗弃 Note 7，会把它留在身边。

另外，也有人说，就算透过 Note 7 看电影两个小时，手机也不会发热，为此深信手机没有问题。也有人说，Galaxy Note 7 的配备让他爱不释手，带着手机就像带着一台Surface Pro 一样，他可以随时修图，随时创作，为此不想更换其他手机，因为市面上没有一台手机比 Galaxy Note 7 更为出色！

Galaxy Note 7 作为三星重磅出击的旗舰手机，一直备受消费者和市场期待，甚至被认为是今年度最佳手机之一，如今因为“一爆再爆”而沦为“爆炸手机”，真是叫人不胜唏嘘！不过，在三星陷入多事之秋时，还有不少铁粉如此忠诚，相信是三星最大的安慰，真的不枉费三星多年来在手机市场上的苦心经营。
