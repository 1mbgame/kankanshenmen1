三星很忙！寄发“特别的盒子”给用户回收Note 7

韩国电子业巨头三星因为 Galaxy Note 7 而忙得焦头烂额！更新版的 Note 7 依然不争气，频传爆炸事故，导致三星电子被迫全面停产、停售，同时禁止用户继续使用 Note 7，启动全球二次召回手机、退款计划，回收这些犹如“炸弹”一般的问题手机。然而，一些快递公司基于安全考量，拒绝运送 Galaxy Note 7，以致三星的召回计划进度缓慢，面对多重阻滞。


图片: https://www.engadget.com


为了让用户顺利的把问题手机寄回给三星电子，并确保托运过程的安全性，三星用尽心思，祭出“防火盒”！三星开始向用户寄出“防火盒”，让用户在安心、安全的情况下退货。

这个“防火盒”不是一般的盒子，内附一个抗静电袋和保护手套。回收盒的特别之处则是，内部由陶瓷纤维纸制成，能够耐热也能承受高温。三星还贴心的为用户提供一张简单的说明单和运输建议，显示回收盒只适合陆运或海运，禁止空运。

图片: http://www.zdnet.com/


根据说明单，用户须先戴上手套，将 Galaxy Note 7 放进防静电的袋子中，再放进较小的盒子，之后再放入内层由陶瓷纤维制成的盒子，才能透过陆运寄回给三星。

图片: http://www.zdnet.com/


根据英国《每日邮报》的报道，美国用户已经从11日开始，陆续收到特制的“回收盒”。至于其他地方的用户，则要耐心等待。
