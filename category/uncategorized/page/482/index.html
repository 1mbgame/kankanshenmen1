荣耀再推高性价比新机！售约RM519

华为旗下的荣耀手机向来以高性价比著称，随着竞争对手相继推出新机后，荣耀也将推出一款性价比高的新机 — Honor 6C！


荣耀 6C 已经现身中国工信部，蓄势待发。根据工信部发出的信息，荣耀 6C 的外形普通，但配置相对其他同等级手机有过之无不及！Honor 6C 采用 5.2 英寸720p 显示屏，搭载 Kirin 659 八核处理器，4GB RAM + 32GB ROM，后置指纹识别，电池容量为 2900mAh，运行基于 Android 7.0 的 EMUI 5.1 操作系统。根据之前曝光的消息，不排除荣耀 6C 将推出 3GB RAM 版本。相机方面，Honor 6C 前置 800 万像素摄像头，后置 1300 万像素镜头。

Honor 6C 提供金色、黑色、蓝色、粉色四种机身配色，据悉 3GB RAM 版本售价 799 人民币 （约 519令吉），4GB RAM 版本售价 999元（约 649 令吉）。
