华为Mate 9开放预定！ RM499蓝牙键盘免费送！


图片取自cnBeta.COM
华为Mate 9刚刚在中国发布，马上就有本地官方消息透露，华为将在11月15日（周二）透过本地两大网购商城LAZADA和VMALL开放预购新手机－－Mate 9！首10名成功订购的买家还可以获得价值RM 499的蓝牙键盘，手快有手慢无咯！
华为Mate 9采用的是5.9吋Super AMOLED屏幕，搭载Kirin 960处理器并且搭配4GB/6GB RAM + 32GB/64GB /128GB ROM 记忆体组合（同时可透过SDcard扩充最高至256GB）！另外，支持后置指纹识别、Type-C接口、双卡双待、VR DayDream，还有4000mAh电池容量，并且支援支持5V、5A电流的超级快充，同时运行基于Android 7.0系统的EMUI 5.0操作界面。

图片取自sogi.com.tw
此外，华为还号称Mate 9用18个月都不会卡顿！这是由于华为独创的Machine Learning智能系统，整个系统具备智能行为预测、智能资源调度、自动内存回收、文件系统优化等功能，可确保流畅、高速的用户体验。
至于相机部分，Mate 9同样和莱卡合作，搭载经LEICA认证 1200万+2000万像素双镜头，支援F2.2光圈、激光/相位/深度/对比度四合一混合对焦、OIS光学防抖，双倍光学变焦等等；前置镜头为800万像素，支援F1.9光圈、自动对焦。根据LAZADA的资料显示，这次率先抵马的是4GB RAM + 64GB ROM版本，售价为2699令吉。明日起正式开放预购，11月22日正式开卖！
