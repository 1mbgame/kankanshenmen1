Pokemon Go Buddy 终于来了！选哪个Buddy好？

Pokemon Go 玩家引颈长盼的 Pokemon Go Buddy 终于登陆马来西亚了！

更新版的 Pokemon Go Buddy 页面基本上没太大分别，惟左下角的玩家头像旁多了一只精灵。点击玩家图像后，右下角也多了一个选项。点击 Buddy 后就可以依据个人喜好选择任何一个喜爱的精灵陪在身边。若是选择体型较大的，精灵庞大的身躯跟在主人后面，貌似很有“安全感”。若是选择体积较小的，精灵犹如小宠物，画风可爱逗趣。

不过，让人略显失望的是，小精灵并不如想象般沿途陪在主人身边，主人走到哪，精灵跟到哪。



Pokemon Go Buddy 允许玩家自行选择所喜爱的精灵陪在身边趴趴走，不同种类的精灵还能够为主人“赚取”不同数量的糖果奖励 (Candy)。这也表示，与其等着某些“鲜少出现”的精灵再次出现，以存够糖果进化，那不如靠自己，把它列为Buddy! 走多一些赚多一些糖果！

值得一提的是，Buddy可以随时更换！不过，由该精灵所累计的步行里数也会跟着报销！所以，千万不要半途“移情别恋”，否则就得 “重新来过”。

根据外国网友，只要带着精灵步行10公里后，精灵就会跳上训练师的肩膀！如果你是Pokemon 铁粉，皮卡丘应该是不二选择吧！说多无谓！赶紧走动走动，累计10公里里数， 让皮卡丘站在你的肩膀上，当个更贴近主人的 Buddy吧！
