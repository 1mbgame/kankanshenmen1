特朗普爆冷胜选后 全球网民都在谷歌热搜什么？

美国总统选举尘埃落定，政治素人特朗普爆冷完胜政治女强人希拉里，顺利入主白宫。基于选举结果出乎意料，让许多人跌破眼镜，全球网民也被“吓坏”，纷纷向 Google 寻求答案，让一些关键字句瞬间荣登最热门搜索关键词榜单。


图片: Reuters


根据 Google Trends，特朗普胜选后，很多人都在谷歌打出和 How did (如何) 有关的字句，其中 “How did Trump win” （特朗普如何获胜）成为最多人搜索的问题！其他登上热搜榜的问题还包括 “How did this happen?” (这如何发生的？) 、“How did Hillary lose?” （希拉里如何败选？）等等，可见网民对特朗普胜选感到万分惊讶，纷纷寻求解答。

图片： https://www.rt.com


除了 How did……  (如何)，其实还有 (Why did……)。字句排行榜中，“Why did Hillary lose” (为何希拉里会输？) 占据榜首、紧随而来的有 “Why did Trump win?” （特朗普为何会赢？)、 “Why did Hillary wear purple?” （希拉里为何穿紫色衣服？）、“Why did people vote for Trump”？(为何人们投选特朗普？) 、“Why did Hillary concede？” (为何希拉里承认败选？)

图片: https://www.rt.com


另外，“如何移民”、“世界末日”、“如何弹劾总统” 等等关键字句的搜寻次数也突然暴升，显示美国人对国家未来的悲观看法，对新任总统缺乏信心。
