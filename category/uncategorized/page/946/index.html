小米Note 2锁定发布日期！找“他”代言聊科技很惊喜！

中国著名手机品牌小米再度重磅出击，将在一个星期后，推出承载小米高端梦的小米 Note 2！暌违了一年九个月，小米这次带来的旗舰手机 Note 2 将采用双曲面设计！到底这款设计的魅力何在？小米宣布将在10月25日举行新品发布会，揭开 Note 2 神秘面纱！

值得一提的是，小米找来“电眼影帝”梁朝伟代言，而且梁朝伟将在新品发布会上，和大家聊科技！到底双曲面的 Note 2 手感如何？梁朝伟将在北京大学揭开谜底。



小米手机官方在微博贴文说，“感谢您1年9个月的等待！10月25日北京大学体育馆，小米 Note2 双曲面新品发布会，梁朝伟现场和您聊聊“一面科技，一面艺术”。

图片: http://www.maizhi.com

小米 Note 2 主打口号“一面科技，一面艺术”，看来Note 2在功能和设计下了一番功夫。根据小米官方自爆的双曲面设计，相信很多米粉都很期待这款与众不同的全新设计。消息指出，小米 Note2 可能搭载骁龙821处理器、5.7英寸 2K 屏幕、超声波前置指纹、双摄像头以及采用三面无边框设计，最大亮点或许是陶瓷机身。至于价格如何？是否也走高端路线？则有待10月25日的宣布咯！
