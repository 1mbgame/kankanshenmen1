八核处理器+3GB RAM！骁龙版Redmi Note 4 大马开卖

如果你想买一部价格低于一千令吉的平板手机 (phablet)，那么 Redmi Note 4 就是最好的选择！搭载 Snapdragon 625 处理器的红米 Note 4 终于在马来西亚发售了！

骁龙版 Redmi Note 4 配有5.5寸 1080p 显示屏、骁龙 625 八核处理器、3GB RAM + 32GB ROM、前置摄像头为500万像素、后置镜头为 1300万像素 (有相位对焦功能、f2.0 光圈)、电池容量为 4100 mAh、双卡双待、运行 Android 6.0 系统、配有高速指纹识别。“大马版”红米 Note 4 有金色和银色供消费者选择。

目前，马来西亚小米还未引进红米 Note 4，但消费者可在 SatuGadget 或 DirectD 上购得，3GB RAM + 32GB ROM 版本售价介于 839 – 849 令吉。
