7月7日重出江湖！Galaxy Note Fan Edition 限量开卖

三星一代机皇 Galaxy Note 7 去年面市不久出师不利，因为电池缺陷引发爆炸事故，三星不得不全面回收，并强调停产停售，不过看来三星还是不甘心让一代旗舰机从此销声匿迹。综合外国媒体报道，经过一连串安全测试的 Galaxy Note 7 翻新版，终于确定将在7月7日重新上市！


翻新版 Galaxy Note 7 将以“Galaxy Note Fan Edition” 之名重出江湖。Galaxy Note Fan Edition 的配置基本上和 Note 7 一样，不同的是电池容量从原本的 3500mAh 减少至 3200mAh，而且系统已经更新，和 Galaxy S8 一样可以使用 Bixby 语音助理。
三星内部人士强调，Galaxy Note FE 出厂前已经过严苛的安全测试。值得一提的是，Galaxy FE 的包装盒与机身背面刻有「Galaxy Note Fan Edition」和“R” 标志，避免消费者混淆。

Galaxy Note Fan Edition 将于7月7日在韩国重新发售，限量40万台，售价为699,600 韩元（约2629 令吉）。目前，可以确定的是，Galaxy Note Fan Edition  不会登陆美国和中国市场，至于亚洲其他市场，则有待三星发布官方声明。
