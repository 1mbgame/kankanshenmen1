发布会倒数一周 三星Note 8完整配置确认！

三星电子将在8月23日正式发布新一代机皇 Galaxy Note 8，也就是说发布会开始进入倒数阶段。如今，三星 Galaxy Note 8 正式在 GFXBench 数据库现身，完整配置提前曝光！



根据曝光的消息，Galaxy Note 8 将采用 6.4 英寸 1080p 显示屏（分辨率为 2960 x 1440），比例为 18.5:9，Note 8 将搭载 Exynos 8895 处理器 (主频为 2.3GHz)，6GB RAM + 64GB ROM，同时配置 800 万像素前置摄像头和 1200 万像素后置摄像头，运行 Android 7.1.1 操作系统。

三星下一代机皇或在9月底正式开卖，预计售价将高达 6500 人民币（约 4200 令吉），而高配版价格或逼近8000元 （约 5100 令吉）。一部手机要价 4000- 5000令吉也太夸张了吧！到底 Note 8 会给消费者带来怎样的惊喜？值不值得这个加码？唯有等 8月23日答案揭晓！
