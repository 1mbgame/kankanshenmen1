Honor首款全面屏手机报到 麒麟659+双摄+长续航，售价有惊喜

全面屏设计无疑是智能手机今年最流行的趋势，各厂商纷纷推出高屏占比的手机，华为旗下的荣耀也正式加入行列，发布旗下首款全面屏手机 — 荣耀畅玩7X！

荣耀7X采用5.9英寸高清全面屏设计，屏幕比例为18:9，分辨率为2160 x 1080像素。外观工艺上，荣耀7X采用一体化金属机身设计，辅以细腻的纳米注塑工艺，背面中央内置指纹识别传感器。

配置上，荣耀畅玩7X搭载Kirin 659八核处理器（主频为2.36GHz），4GB RAM + 32GB/ 64GB/ 128GB ROM（可透过MicroSD卡扩展至最高256GB）。荣耀畅玩7X电池容量为3340mAh，支持快充3.0。官方宣称待机可达22天，并可播放视频长达12小时，上网可达12小时。值得一提的是，荣耀畅玩7X还加入创新的视频增强技术与“一键分屏”技术，通话除了能在观赏视频时看到更多细节，还能自动分屏让多个APP同时操作。

相机方面，荣耀畅玩7X后置1600万 + 200万像素双摄像头，前置800万像素自拍镜头。前者支持大光圈、背景虚化人像模式，后者也不遑多让，同样支持背景虚化，还支援性别识别美颜以及变妆特效等功能。

荣耀旗下产品向来以高性价比著称，Honor畅玩7X 32GB版本售价1299元（约829令吉），64GB版售价1699元（约1089令吉），128GB版售价1999元（约1279令吉），从10月17日开始线上线下开启首销！好消息是，马来西亚也在首销名单之列！
