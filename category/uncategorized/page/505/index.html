为何Note 8电池容量比S8+少？原来暗藏“电池大革新”

三星新一代旗舰机 Galaxy Note 8 正式推出后，大家除了将他与 Galaxy S8/ S8+ 作比较，也格外关注新机的电池容量与安全性，毕竟 Galaxy Note 7 因为电池过热引发爆炸的阴霾依然挥之不去。Galaxy Note 8 在电池技术与安全性上有什么突破革新？三星如何借此挽回消费者的信心，是 Galaxy Note 8 肩负的重要使命之一。


与 Galaxy S8+ 相比，Galaxy Note 8 的电池容量确实缩水了，Galaxy S8+ 搭载 3500mAh 电量，而 Galaxy Note 8 的电池容量减至 3300mAh。尽管差别不大，但人们的疑虑仍在。三星电子移动通信总裁高东真接受海外媒体访问时，对外解释了 Galaxy Note 8 降低电池容量的原因，原来这其中藏着“电池大革新”！

高东真解释，三星 Galaxy Note 8 采用10nm制程的处理器，能将手机的能量效率提升 30%，而且用户可以自行调节合适的电池模式。他还强调，Galaxy Note 8 使用两年后，依然能保持出厂95%的容量，也就是说，电池在2年后的损耗率只有5%。

销量方面，高东真则期待 Galaxy Note 8 能够卖出超过 1100 万部，突破 Galaxy Note 5 的销量。虽然分析普遍认为，Galaxy Note 8 并不如 Galaxy S8/S8+ 让人惊艳，但各方面却是无可挑剔。随着 Galaxy Note 7 出师不利而对三星造成的连带影响，不难理解三星宁可保守稳重也不愿哗众取宠的做法。
