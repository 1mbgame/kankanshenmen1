用户"冥顽不灵"不愿退回Note7！三星使出“杀手锏”应对

三星一代机皇 Galaxy Note 7 出师不利，疑似因为电池过热引发自燃爆炸现象。三星基于安全考量启动全球召回计划，并宣告停产、停售 Note 7，一代旗舰手机就此“走上末路”。尽管如此，仍有不少手机仍在市场上，还未顺利回收……


图片摘自 YouTube

一些没有遭遇电池问题或对三星“忠心耿耿”的 Note 7 用户，不愿服从召回计划，叫三星无可奈何。最终，三星忍无可忍了，决定使出“杀手锏”，逼迫用户彻底放弃 Note 7！三星将在15日推送“死亡系统更新”，彻底终止手机的充电功能！只要用户一更新系统，Galaxy Note 7 电池耗尽后就无法再使用了。

与此同时，三星也会在特定地区禁用 Note 7 蓝牙和 Wi-Fi 功能，让用户无法连接移动网络，丧失智能手机的功能。根据了解，单是美国地区，三星仅召回 85% 的 Note 7，还有超过 28万部 Note 7 仍在市场上。

图片: http://venturebeat.com/
三星早前已经通过软件更新，限制 Galaxy Note 7 只能充电 60%，还不断发送提醒要求用户退回手机，不过，召回速度缓慢，最后迫使三星使出“终极招数”，推出“死亡系统更新”。
