小米平板3要来了！最低售价或RM832

小米官方上个月底曾经暗示新一代平板将在近期发布，最新消息指出，小米将在4月11日举行新品发布会，同台发布小米6和小米平板 3/3 Pro。小米平板已经将近两年没更新了，新平板让人期待，作为安卓阵营中表现稍微出色的平板，小米平板3预计会推出两个版本，两者将搭载不同尺寸的屏幕还有截然不同的系统！

图片摘自网络

根据中国媒体报道，有电商悄悄上架还未发布的小米平板 3，导致售价和相关配置率先曝光。消息指出，第一款小米平板3将沿用上一代的设计，采用 7.9英寸屏幕，搭载 4GB RAM + 64GB/128GB ROM。这款平板将运行 Android 7.0 Nougat 系统，64GB 版售价 1299人民币 (约 832 令吉)，而 128GB 版本的售价为 1599 人民币 (约 1025 令吉)。

图片摘自网络

至于小米平板 3 Pro，则可能采用 9.7 英寸屏幕，分辨率为 2048 x 1536，搭载 4GB RAM + 64GB/128GB ROM，不过不是运行安卓系统，而是配置 Intel 8750 处理器，运行 Windows 10。64GB版小米平板 3 Pro 售价为 1699 人民币 (约 1089 令吉)，而 128GB 版本的售价为 1999 人民币 (约 1281 令吉)。
