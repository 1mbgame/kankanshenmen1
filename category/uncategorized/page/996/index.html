三星急于扭转乾坤 S8 型号抢先曝光？

三星旗下的 Galaxy S7 是S系列中最受欢迎的旗舰手机之一，可谓协助韩国企业改善财务状况的其一功臣。随着 Galaxy Note 7 因为电池引发爆炸事故接二连三，Galaxy Note 7 的安全隐患让消费者信心猛挫，让三星陷入尴尬局面，非得想方设法扭转乾坤。

在这种情况之下，三星急需一款全新旗舰设备恢复市场信心，同时弥补召回 Galaxy Note 7 所导致的庞大亏损。其中一项有助于扭转局势的妙招，便是提早推出 Galaxy S8！


图说: http://www.geeky-gadgets.com


外国媒体 SamMobile 独家曝光三星 S8 型号手机，据悉，Galaxy S8 有两个版本，分别代号为 Dream 和 Dream 2，型号为 SM-G950 和 SMG-955 。一般猜测，首款为 S8, 第二款即是 S8 Edge。


Galaxy S7 型号是 SM-G930，如果延续三星一贯作风，每代手机的型号以10来递增，那么 S8 应该是 SM-G940 才对。但事情并非如此。原来，韩国人也忌讳代表不吉利的“4” 。其实，这也不是三星首次回避 “4” 这个数字了。Galaxy S3 型号为 GT-I9300，但 S4 型号却是GT-I9500。然而，Galaxy S4 并没有被“牺牲”而直接取名 S5。

根据外媒 Digitalmusicnews 爆料，三星也可能步苹果 iPhone 7 后尘，取消 3.5mm 耳机插口，并以三星专属的接口取而代之。也有消息指出，S8 或 S8 Edge 可能内置4个摄像头。
