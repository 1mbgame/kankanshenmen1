北马水灾严重 Facebook罕见启动这项功能

北马槟城与吉打水灾灾情严重，大山脚部分地区的水位久久不能退去，灾情堪忧。此次水灾严重，以致面子书启动“安全通报”功能，让北马用户向亲友报平安。


图片: https://www.nst.com.my

面子书从2014年开始，在全球启动“安全通报”功能，不但允许用户向外界报平安，也允许用户在灾难发生时寻求或提供帮助。位于事发地区的面子书用户可能会收到通知，提醒用户到有关网站报平安。


早在这项功能启动时，并没有专属网站，如今面子书已经设立相关平台，让用户按下“I’m  Safe”就能报平安。这个网址不但提供有关水灾的相关信息，也让用户知道哪一区的灾民寻求协助及哪些单位提供援助。另外，用户还可以查看与你身处同一个地区的亲友是否平安，可以向对方发送“Ask If safe”讯息，关心对方的处境。如果你是槟城子民，赶快登陆危机应对页面向亲友报平安或关心其他亲友的情况吧！
