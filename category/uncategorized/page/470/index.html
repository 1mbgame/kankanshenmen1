Galaxy Note 8全球开卖 韩国首日销量实在惊人！

三星日前推出全新机皇 Galaxy Note 8，作为三星首部搭载双摄的旗舰机，Galaxy Note 8 配置全视曲面屏，拥有极高颜值，配置规格高档不愧为旗舰。从上周四开始，三星在全球各地启动预购活动，向来对三星支持度高的韩国民众以行动力挺本土产品，Galaxy Note 8在短短一天内的预定量惊人！



截至周五，单是韩国本土的预定量高达39.5万台，突破40万台不是难事。作为 Galaxy Note 7 的“继任者”，Galaxy Note 8 用一天时间刷新 Galaxy Note 7 18天才达到的傲人成绩！这成绩足以说明，三星已经走出 Galaxy Note 7 事件的阴霾，成功挽回本土消费者的信心。数据显示，高达65％的消费者预购64GB版本，蓝色突围而出成为最受欢迎的角色。值得一提的是，Galaxy Note 8（64GB版）在韩国的售价为109万韩元（大约 4050令吉），而256GB版本的售价高达125万韩元（大约 4600令吉），不管是哪种版本的售价都已经突破4千令吉，与传言中的iPhone 高价相去不远！

由此可见，Galaxy Note 8 绝对是 iPhone 8 的劲敌，虽然 Galaxy Note 8 在取得先发优势，但别忘了，【十周年纪念版】的iPhone 8 即将在下周亮相，到底谁更胜一筹，iPhone 8 的出现会否影响 Galaxy Note 8 的销量？Galaxy Note 8 能够达到官方预期，超越 Galaxy Note 5 1100万台销量目标？时间与数据将是最好的证明，一场激烈的竞争一触即发。
