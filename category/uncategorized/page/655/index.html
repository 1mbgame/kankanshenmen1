【全球爆发勒索病毒】大马也遭受威胁？关键四招教你防护Windows

从星期五开始，全球包括欧美和亚洲等地约150个国家的电脑系统先后遭黑客攻击。欧盟警署负责人宣称，至少20万部电脑受害，威胁范围正在扩大，预料周一人们上班打开更多电脑时，勒索病毒会卷土重来，“中招”电脑数目将进一步攀升。

名为 WannaCry 的勒索病毒主要针对使用微软视窗系统的电脑。用户电脑中毒后须支付价值约 300 美元的比特币才能解锁，赎金每隔数小时便「加码」，若用户不付钱，七日后将无法读取电脑内的文件。如果真的遭受袭击，就真的是 WannaCry “想哭”了！


图片摘自网络
大马通讯及多媒体委员会 (SKMM) 已经就此事回应，宣称我国至今还未遭受勒索病毒攻击，也没有接获任何勒索病毒攻击的投报，但会密切观察事态发展。SKMM 呼吁用户更新最新版的电脑系统，善用防毒软件，同时备份电脑里的重要资料。

那要如何防范电脑遭 WannaCry 勒索病毒攻击？

一、及时更新Windows 发出的更新软件
【非常时期，采取非常手段】为了保护所有的 Windows 用户不受勒索病毒攻击，微软特别发出更新软件，为使用更早期软件的用户，包括 Windows XP、Windows 8和 Windows Server 2003 提供紧急更新档。微软也鼓励用户未来定期更新系统，以降低中毒风险。Windows10 的用户暂时可以放心，因为不是这次恶意攻击的目标。

二、善用防毒程式
【招数不怕老，关键是有效】防毒程式可以防止勒索软体下载到电脑，也能侦测勒索软体的存在。多数防毒程式可以扫描文件，在下载前侦察文件是否涵盖勒索软体，并能在用户浏览网页时，及时阻止恶意广告偷偷下载恶意软体。Windows Defender 已经能够有效侦测并清除发作中的恶意程式，使用者可自行下载。

三、备份档案
【不怕文件消失，只怕没有备份】遭遇勒索病毒攻击最大的损失是硬盘彻底清空、档案消失不见，不管是文件、照片还是其他重要资料。最佳的防范方法就是在断网的情况下，将重要资料备份到另一个独立的系统，让网络勒索病毒无法得逞。

四、对陌生电子邮件、网站、APP、链接高度戒备
【不熟悉的不要碰，网络陷阱满是坑】用户必须具备网络安全意识，陌生链接不点击，陌生文件也不要下载，不熟悉的邮件更不要打开。因为黑客一般透过这些链接，让用户在不知情的情况下，下载恶意软体到电脑上。

目前，英国、意大利、俄罗斯、亚洲等全球150个国家的电脑遭受勒索病毒攻击，黑客主要针对车厂、医院、商店和学校的电脑。其中，英国国家医疗服务体系遭遇大规模网络攻击，英格兰、苏格兰多家公立医院的电脑系统几乎同步瘫痪，电话线路也被切断，不少急诊病人被迫转送到其他医院。中国方面，部分高校的电脑难以幸免，学生的毕业论文被病毒加密，必须支付高额赎金才能解密。
