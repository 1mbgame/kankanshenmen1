堪称【华为性价比最高双摄手机】Honor 6x 情人节在大马发售！

向来主打高性价比的华为荣耀系列深受消费者欢迎，如今再有一款 Honor 系列手机即将登陆我国，说的就是 Honor 6x！



根据 Mdroid 报道，华为确定将在2月14日情人节当天在我国发布 Honor 6x，预计上市的会是 3GB RAM + 32GB ROM 版本，售价料为 1199令吉。目前，华为官方还未正式宣布”大马版” Honor 6x 的价格、颜色等详情，花粉们耐心等待吧！

Honor 6x 配备 5.5寸全高清屏幕、搭载麒麟 655 八核处理器、内存和存储空间提供三个选择: 3GB RAM + 32GB ROM、4GB RAM + 32GB ROM、4GB RAM + 64GB ROM，支持 MicroSD 卡扩展，电池容量为 3340mAh。摄像头方面，Honor 6x 配置 200万像素 + 1200 万像素后置双镜头，前置 800万像素摄像头。
