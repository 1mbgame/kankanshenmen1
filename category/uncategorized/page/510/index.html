24日接受预购 三星Galaxy Note 8豪送好康！

倒数两天！三星新旗舰 Galaxy Note 8 即将正式亮相！三星电子将在我国时间23日晚间11点，在美国纽约举行新品发布会，隆重推介下一代机皇 Galaxy Note 8。爆料向来准确的爆料大神 @evleaks (Evan Blass) 这回捎来好消息！

图片: Evan Blass 推特


根据 @evleaks 爆料，凡预购三星 Galaxy Note 8 的消费者，将获得额外好康！三星将加赠一张 256GB 的 microSD 记忆卡 + 无线充电器，或 360 °相机，二选一。那三星何时开放预购？爆料大神说，就在8月24日！也就是新品发布会的隔天就接受预购，9月15日正式发售！如果消息属实，三星电子这次倒是诚意十足，不过马来西亚是否在首发名单内，开放预购日期与赠品是否和海外一样？则有待大马三星官方宣布。


另一方面，国外媒体报道指称，三星 Galaxy Note 8 价格将会飙升，中国售价随之曝光。据悉，64GB 版本售价 6288 人民币 （约 4040 令吉），128GB 版本要价 7088 人民币 （约 4550 令吉）；256GB 版本售价 7988元 （约 5150 令吉）。
 

综合此前曝光的消息，三星 Galaxy Note 8 会采用 6.3 寸 QHD+ SuperAMOLED 曲面显示屏，Snapdragon 835 处理器，6GB RAM + 128GB ROM，后置双摄像头，Bixby AI，3D 识别功能，后置指纹识别，IP68 防尘防水功能等，而且传言指出，Galaxy Note 8 还会一口气推出 8 种配色！



三星韩国官方 YouTube 也释出两支预告影片，相信 S Pen 和双镜头将是三星 Galaxy Note 8 的最大卖点。
