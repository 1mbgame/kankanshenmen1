三星翻盖手机视频曝光！发布日期确定

三星旗舰翻盖手机W2018早在5月份通过中国工信部认证，配置与真机随后也在互联网上曝光，如今更有网友在微博晒出一段三星翻盖手机视频！虽然手机的来源暂不清楚，但从视频中可以看到，翻盖手机采用三星最近的设计理念，比如后置指纹扫描器内置于相机传感器旁边。


除了视频曝光之外，三星W2018正式发布日期也终于敲定。根据官方释出的海报，三星将在12月1日，在一年一度的“心系天下”活动中正式发布W2018，韩磊、曾宝仪、崔健等明星艺人也会参与其盛。


综合早前曝光的传闻，W2018配置内外两个4.2英寸全高清触控屏幕，采用骁龙835处理器，辅以6GB RAM + 64GB/128GB ROM，前置500万像素镜头，后置1200万像素摄像头，电池容量为2300mAh，预计运行Android Nougat操作系统。
