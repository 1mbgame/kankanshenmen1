三星铁粉拒还Note 7！电讯商采取“极端措施”对付

一代机皇 Galaxy Note 7 出师未捷后，三星全球召回手机，并宣布停产、停售。不过，至今仍有不少三星铁粉拒绝归还 Galaxy Note 7，迫使三星采取多项措施，包括推送相关系统，让手机无法充电，更和移动运营商合作，让 Galaxy Note 7 彻底断网，以此逼迫用户归还手机。


图片: http://www.droid-life.com/

尽管三星使出的各项“杀手锏”取得一定的效果，但无数铁粉依然无视官方呼吁，拒绝归还 Galaxy Note 7。根据外国科技媒体报道，美国电讯商 Verizon 为此采取极端措施，强制转接用户的所有电话到 Verizon 的客户服务部门。Verizon 这么做是希望用户赶紧退还 Note 7 或更换其他手机，避免 Galaxy Note 7 为用户和身边的人构成威胁。

根据统计，三星总共出售 430 万部 Galaxy Note 7，但迄今只召回大约 400万部。单是韩国就出售近 95万部 Note 7，但仍有 5万部还未退还。三星电子将在23日举行新闻发布会，对外公布 Galaxy Note 7 爆炸的原因。
