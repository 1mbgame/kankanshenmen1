HTC Ocean Note“来势汹汹” 功能或超越谷歌Pixel

HTC 将在明年1月12日发布代号为 Ocean Note 的最新旗舰产品，如今距离发布会越来越靠近了，有关 HTC Ocean Note 的细节和特色也浮出台面。


截图自YouTube

根据台湾媒体 eprice 报道，HTC Ocean Note 可能完全取消实体按键，首次采用曲面屏设计，搭载骁龙 821 处理器，同时取消3.5mm 耳机接口。除此之外，HTC Ocean Note 的一大亮点便是相机，HTC 料将大幅提升 Ocean Note 摄像头的表现，据称拍照功能将会超越谷歌 Pixel，有望获得著名相机评测网站 DxOMark 的最高评价。HTC 此举貌似仿效谷歌，谷歌推出 Pixel 前，先将手机送到 DxOMark 测试，然后在发布会上宣布 Google Pixel 是相机评测网站公认的最佳拍照手机。

截图自YouTube

消息指出，HTC Ocean Note 将会支持 USB Type-C 接口，并追随 HTC 10 evo 脚步，附送 USB Type-C 耳机，也能支援 Adaptive Audio，依据用户的耳道结构调整音频。外形方面，新旗舰的显示屏可能达到 6.1 英寸，若消息属实，那么屏幕分辨率为 2K 的可能性较高，至于是否如传言所说，配有边缘触控功能，则有待进一步消息。

台湾媒体还爆料，HTC 还会带来两款新机，分别为走中端路线的 HTC ONE X10 和定位中低端的 Desire 系列手机。
