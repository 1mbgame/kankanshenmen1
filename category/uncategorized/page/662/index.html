报章为S8广告"豁出去"！创意堪称一绝但被读者嫌弃

刚刚面市的旗舰机 Galaxy S8 可谓三星的“救命稻草”，三星希望借助这款产品消除 Galaxy Note 7 爆炸事故对手机品牌带来的负面影响。为了加大 S8 的宣传气势，三星不惜重金推广和营销全新的旗舰机。登报宣传是三星宣传新机的手法之一，但这次三星并非“好好的”在报纸上登广告，而是采取大胆创意的手法，把整份报纸变成一则广告，成功掀起话题。


这份报纸是瑞士规模最大的德语日报 《20 minuten》，该报为了三星广告而罕见的全幅改版！和一般广告不同的是，原本中规中矩彩色图文排版的报纸，封面出现像似随意涂抹的蓝色颜料，读者若继续翻阅报章，就会发现内页出现各种颜色的透明色块。三星成功透过平面设计巧妙融入 S8 推崇的设计理念，让整份报纸看起来更像一本设计杂志。

虽然三星的广告吸引眼球，不过引起两极反应，广告界普遍赞赏三星大胆又创意，但也有读者不买账，认为广告影响阅读体验。读者纷纷到《20 minuten》的官方脸书留言，表示 “阅读体验很糟糕”、“几乎看不清楚报章上写什么”、“我想要的是一份报纸”。有鉴于此，三星应该是唯一也是最后一个能在有关报章上打类似广告的厂商了，因为《20 minuten》已经就此事向读者道歉，强调不会重蹈覆辙。

随着创意广告引起关注后，为三星设计广告的著名广告设计师 Jung von Mat 和卢塞恩设计工作室 Studio Feixen 也被起底。Jung von Mat 曾为 Mercedes Benz 策划创意短片而获奖，而 Studio Feixen 的三名独立设计师则是《纽约时报》、《连线》等杂志封面设计师。对于设计师的创意，你有什么看法？
