Note 7 改良了还是有问题？ 三星: 与电池毫无关系！

三星最近陷入多事之秋， Galaxy Note 7 疑因电池过热引发爆炸的事故接二连三。该公司已经宣布全球紧急召回、换机计划，但事情没有想象中顺利，部分消费者抱怨，换机而来的改良版 Note 7 依然存有发热和耗电等问题，让“电池”再度成为众矢之的。

三星公司发言人驳斥有关指控，强调改良版手机出现的问题“和电池毫无关系”，宣称那只是大量生产线中出现的个别案件。


图片: http://venturebeat.com

一名韩国消费者早前投诉，指称改良版 Galaxy Note 7 耗电速度惊人，还无法正常充电，就算彻夜充电，电量也只有10%。美国消费品安全委员会监督机构就 Galaxy Note 7 爆炸门发表谈话时指出，Note 7 之所以会爆炸，是因为三星强行在手机窄小的空间置入太大的电池，引发短路才会爆炸起火。

韩国三星电子宣布，基于全球召回换机计划未完成，为此原定 9月28日让 Galaxy Note 7 在韩国重新上架的计划不能如期进行，暂时宣布推迟三天，最快10月1日重新发售。三星披露，韩国本土约有40万人购买 Galaxy Note 7，其中一半已经响应召回计划。
