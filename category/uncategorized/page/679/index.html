要来的始终会来！Note 7 易名回归价格"出乎意料"

三星Galaxy Note 7 因为电池过热频传爆炸事故，迫使三星全球召回手机并停产停售，让一代机皇黯然退场。之前已有报道指出，三星将出售翻新版 Galaxy Note 7，把损失减到最低，现在韩国媒体更是进一步确认，Note 7 将会易名回归市场，而且价格让人意外。


图片: http://www.androidauthority.com

根据韩国媒体报道，Note 7 翻新机已经通过 Wi-Fi 认证，并将在6月份从韩国市场重新出发，预计推出30万部手机。不过，价格方面就不如传言说的“便宜一半”，发售价格为 620 美元，也就是大约 2,700 令吉左右！这也表示，翻新的 Galaxy Note 7 只是折扣 30% 而已，相较去年开卖时的 870 美元 (约 3785 令吉) 便宜 250美元 (约 1100 令吉）。

虽然三星还未正式对外公布实际的市场价格，但一般推测，三星不会便宜甩货。值得关注的是，Galaxy Note 7 不会以原名回归市场，而将易名 Galaxy Note 7R。根据了解，三星全球召回了 300万至 400 万部“问题手机”，但预料重新上架的仅有10%，大约 30万部。

图片: http://bgr.com

三星召回 Galaxy Note 7 后调查发现，Note 7 死症在于电池，所以翻新机不得不更换电池。Galaxy Note 7R 内部硬体规格和 Note 7 一样，但电池容量料从 3500mAh 更换至更小容量的 3200mAh 或 3000mAh。官方早前强调，翻新机已经通过8项质量测试，确保万无一失。
