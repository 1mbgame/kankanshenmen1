翻新版Note 7重新上架 受追捧还是遭唾弃？

三星一代旗舰机 Galaxy Note 7 因为电池过热引发爆炸而出师不利，去年推出市场不久就全球召回，停产停售。不过，机皇并没有因此销声匿迹，通过无数安全测试的翻新版 Galaxy Note 7 易名 Galaxy Note Fan Edition，于7月7日重出江湖。大家不免好奇，Galaxy Note 7 销量如何？


根据《韩联社》报道，Galaxy Note FE 重新上架后受到韩国消费者热烈追捧，销量火红。当地运营商9日数据显示，Galaxy Note FE上市当天，韩国三大移动运营商号码移动服务使用件数超过2万件，8日上升到2万4千件。韩国消费者更换新手机时，必须前往移动运营商设置号码转换，营运商以此数据推断，Galaxy Note FE 受到消费者追捧，韩国最近几个月的号码移动服务平均使用件数为1万5千件，Galaxy Note FE 上市那天的号码移动服务使用件数达2万4千件，接近韩国通信委员会制定的市场过热标准。Galaxy Note Fan Edition 7月7日在韩国重新发售，限量40万台，三星至今未发布销量数据。

Galaxy Note FE 售价为69万韩元，约马币 2600 令吉，相较 Galaxy Note 7 便宜大约 30万韩元（约 1125 令吉）。
