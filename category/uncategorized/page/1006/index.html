《超级马里欧》或有Android版 坐等上线时间

经典游戏《超级马里欧》 Super Mario Run 在苹果发布会上，可谓 “喧宾夺主” 一度抢走iPhone 7 的风头。是的，任天堂（Nintendo）的《超级马里欧》游戏将会登陆苹果iOS，为果粉和忠实玩家带来惊喜。

不过，安卓 (Android) 用户也无需感到失望。根据美国高客传媒 (Gawker Media) 旗下的知名电子游戏网站Kotaku，任天堂未来有意推出Android版的《超级马里欧》！至于什么时候上线，目前还是未知数，不过一般相信，会在苹果App Store上线以后。



“Super Mario Run”是任天堂首个正式推出的手机游戏，除了单人模式，用户也可以和世界各地的玩家一较高下。
 
超级马里欧之父宫本茂昨日在苹果发布会上指出，这款新游戏将会在年底前，也就是圣诞节期间上线，并提供一次性购买方式，之后不需再额外付费。
