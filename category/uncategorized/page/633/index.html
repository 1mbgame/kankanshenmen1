不把苹果放在眼里 朝鲜推出这样的"iPad"！

苹果公司推出的平板电脑名为 iPad，早已深入民心，但最近朝鲜一家科技公司 Ryonghung 竟然无视苹果的注册商标，将新推出的平板电脑命名“iPad”！


不过，朝鲜公司推出的 “iPad” 和苹果的 iPad 有着天壤之别，虽然朝版 iPad 可用于阅读新闻、处理文件档案，也预装40多个APP，包括计算机、健康、医疗和农业应用程序，但论配置还是相当落后。

图片摘自网络

朝版 iPad 搭载主频为 1.2GHz 的四核处理器，1GB RAM + 8GB ROM，兼容 HDMI 接口，还随机赠送一个键盘，看起来适用于儿童学习。这款平板电脑还可以上网，不过仅能连接朝鲜境内的网站。

图片摘自网络


其实，这不是朝鲜公司首次“抄袭”苹果产品，但直接沿用一模一样的名称还是第一遭。早在2015年，朝鲜科技公司 Blue Sky 就曾推出类似 iMac 的山寨版电脑，而且电脑运行的还是朝鲜自主研发，但看似致敬苹果 Mac OS X 的红星 3.0 操作系统。对于朝鲜版的 “iPad”，有人说“返朴归真”也不错，有人则认为，这样的配置根本是“落后到不行”，你呢？怎么看？
