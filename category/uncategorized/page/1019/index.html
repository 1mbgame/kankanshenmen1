研究3年却“无疾而终”！Google Ara 手机项目喊停

智能手机当道的时代，全球手机品牌急起直追，推陈出新。网络搜索引擎巨头谷歌原先也准备攻入市场，分一杯羹，推出Google模组化智能手机计划“Project Ara”。不过，这项研究了三年之久的计划却证实中途喊停，Ara 手机不会推出市场！

谷歌发言人引述硬体主管瑞克奥斯特洛 (Rick Osterloh) 的决定指出，谷歌决定终止Ara计划，作为精简硬体开发项目的一部分，未来将把重点放在其他重要项目。无论如何，谷歌发言人透露，谷歌可能和合作伙伴签订协议，让第三方将谷歌耗费三年心血开发的Ara计划推出市场。


图片来源: http://www.theverge.com


Project Ara是谷歌近年来重要的项目之一，允许消费者根据本身喜好自由定制、替换或移除任何零组件，就像自由下载应用程序或组建乐高积木一样简单。这么一来，任何故障或过时的零组件，都可以随意撤换，有助于延长设备的生命周期。

图片来源: http://technews.tw


谷歌5月份曾在 Google I/O 开发者大会上捎来Project Ara 开发版将在今年第三季推出市场的好消息，引起市场关注，并放眼2017年推出正式版。不过，随着谷歌态度180度大转变后，这款有别于一般的智能手机计划暂时“无疾而终”。
