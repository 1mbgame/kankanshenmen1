三星公布Note 8发布日期！大屏旗舰比iPhone 8更早亮相

三星 Galaxy Note 7 因为电池过热爆炸而“出师未捷”之后，大家都把焦点放在 Galaxy Note 8 身上。虽然三星机皇还未完全摆脱“爆炸”阴影，但不少 Note 系列粉丝仍热切期盼 Galaxy Note 8 的到来。对于 Galaxy Note 8 的发布日期，坊间消息听多了，消费者半信半疑，但这次轮到三星自行公布日期，绝对可靠！


三星官方在5个小时前于推特正式宣布，下一代“机皇”Galaxy Note 8 将在 8 月23日正式亮相！看来是想在苹果推出 iPhone 8 之前先抢攻市场。三星发布的宣传海报打着“做更大的事情”(Do bigger things) 的宣传口号，显示 Galaxy Note 8 将配置更大的全视曲面屏，图中也出现三星 Galaxy Note 8 与手写笔的身影。

图片: bgr.com

综合此前的传闻，Galaxy Note 8 外形方面预计会借鉴 Galaxy S8/ 8+ 的设计元素，包括搭载无边框的 Infinity 屏幕，但屏占比更高，并会配置康宁大猩猩玻璃5防护，搭载骁龙835或 Exynos 8895 芯片组，4GB / 6GB RAM + 64GB / 128GB ROM，运行 Android 7.1.1 牛轧糖系统，3300mAh电池，支持快充。相机方面，则会首发双摄，后置 1200万双摄像头。到底下一代机皇有多厉害？会带给消费者什么样的惊喜？价格如何？8月23日自有分晓。
