华为一口气推三款笔电 MateBook X 屏占比高达88%

华为23日在德国召开新品发布会，一口气推出三款 MateBook 笔记型电脑 —- MateBook D、E、X。D 系列定位商务影音、E 系列强调二合一、X 系列主打轻薄和颜值。

华为 MateBook X 


从外观而言，Matebook X 和传统的笔记型电脑差别不大，特别之处就是边框和厚度。MateBook X 采用13英寸2K分辨率屏幕，边框仅有4.4毫米，屏占比高达88%，号称世界上最小的13寸笔记型电脑。MateBook厚度为12.5mm，比苹果新版MacBook薄6mm，重量为1.05kg。


MateBook X 搭载英特尔第七代酷睿i7/i5处理器，8GB RAM + 256GB/ 512GB ROM，内置空间智冷技术，在不需要风扇的情况下仍可高效散热，同时配置 41.4Wh的电池，可持续10小时播放视频、办公或上网。值得一提的是，MateBook X不但采用指纹识别与电源键合一的设计，还和杜比公司合作，引入全景声音响系统，使得 Matebook X成为全球第一台拥有 Dolby Atmos’Sound System的笔记型电脑。键盘做到可支持防水溅，充电插头也可为手机充电。MateBook X 提供流光金、深空灰、玫瑰金三种配色，价格介于 1399 欧元 (约 6750 令吉) 至 1699 欧元（约 8155 令吉）。

华为 MateBook E 

MateBook E延续第一代MateBook的二合一设计，是笔记型电脑，也是平板电脑，可在两者之间随时切换，屏幕开合角度最高达到160°。相比上一代，厚度减小至6.9毫米，重量为640g。MateBook E采用12英寸2K屏幕，搭载英特尔第七代酷睿i5处理器，4GB/ 8GB + 128GB/ 256G，标准版采用第七代酷睿m3处理器。


价格方面，m3/4GB/128GB SSD版售价999欧元（约 4800 令吉），i5/4GB/256GB SSD为1199欧元（约 5760 令吉），i5/8GB/256GB SSD为1299欧元（约 6245 令吉）。
华为 MateBook D

MateBook D属于商务影音笔记型电脑，边框仅有6.2mm，厚度为16.9mm，重量为1.9kg。 MateBook D采用15.6英寸屏幕，分辨率为1080p，屏占比高达83%。MateBook D 搭载英特尔第七代酷睿i7/i5处理器，内存最高达16GB，采用256GB SSD固态硬盘+ 1TB HDD机械双硬盘设计，配置 NVIDIA 940MX显卡以及43.3Wh电池，支持杜比全景声，支持8.5小时视频播放。

售价方面，i5/8GB/1TB HDD/NV 940MX为799欧元（约 3850 令吉）；i5/8GB/128GB SSD/1TB HDD/NV 940MX为899欧元（约 4320 令吉）；i7/8GB/128GB SSD/1TB HDD/NV 940MX为999欧元（约 4800 令吉）。
