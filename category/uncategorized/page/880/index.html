Note7 为何爆炸？调查结果: 错不在电池！

三星今年8月推出今年度旗舰手机 Galaxy Note 7，但没想到一代机皇出师不利，相继传出多宗爆炸事故，最后迫使三星做出“最坏的打算”，全球召回手机之余，还宣布停产、停售，让 Note 7 成为寿命最短的旗舰手机。然而，手机为何会爆炸？三星电子积极调查，不过至今仍未公布调查结果，爆炸原因仍是个谜。


图片: http://www.forbes.com/

尽管各界包括三星都怀疑罪魁祸首是 SDI 电池，不过从第三方调查结果看来，SDI 电池是“吃了死猫”。根据第三方承包商 Instrumental 针对 Galaxy Note 7 电池爆炸展开的调查报告显示，爆炸根源不在电池，而是手机过于“激进的设计”存有风险！

报告指出，Note 7 的内部设计过于激进，主要归咎于电池仓的空间不足。事实上，Note 7 的电池和主板之间最窄的部分仅有 0.1mm 的距离，导致电池的+和-在日常使用中受到频繁的挤压和摩擦，进而导致电池起火爆炸。

图片: https://www.instrumental.ai

三星的工程师在设计 Note 7 时精益求精，增加电池的厚度，让设备拥有更大的电池容量，续航力更高。不过，加工的保护罩并没有发挥功效，未能阻止电池受到其他内部组件戳压，最后才会“事与愿违”。

Instrumental 是第三方承包商，主要协助硬体公司修复生产工艺、零部件质量、装配过程和设计缺陷。
