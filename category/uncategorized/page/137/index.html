三星首款“双摄机皇”正式登场！Galaxy Note 8亮点不仅是双镜头

韩国电子巨头三星在我国时间23日晚上11点于美国纽约举行新品发布会，隆重其事推介新旗舰手机 Galaxy Note 8！这是 Galaxy Note 7 因为电池过热引发爆炸，迫使三星全球召回并宣告停售停产后，Galaxy Note 系列重新出发的重要时刻！这部旗舰机还未推出已经未发先热，各种消息、爆料、传言排山倒海而来，到底备受瞩目的三星机皇带给消费者怎样的惊喜？



三星本次发布会的主题为 “Do bigger thing” (大有可为)，顾名思义，手机显示屏延续 Note 系列主打的大屏幕。Galaxy Note 8 显示屏采用与 Galaxy S8 一样的 Infinity Display 无边框超高清屏幕，但尺寸增加至 6.3 英寸，屏占比提升至 83.6%，整体机身尺寸比 Galaxy S8 稍微大一些。


Galaxy Note 8 是三星旗下首款搭载双镜头的智能手机，相信也是当今市场上唯一两个镜头都具备OIS 光学防震机能的智能手机。Galaxy Note 8 后置1200 万像素双镜头，广角镜头配备 f/1.7 光圈、Dual pixel 相位对焦功能，而长焦镜头则配备 f/2.4 光圈、自动对焦功能；前置 800 万像素摄像头，支援 f/1.7 大光圈。值得一提的是，Galaxy Note 8 加入不少拍照新功能，包括拍完照后可以调整模糊程度，还可以在拍照时同步使用二个镜头，做到一次拍摄获得一张广角与一张长焦的照片。

除了双摄之外，Galaxy Note 8 的另一个亮点就是 S-Pen，作为 Note 系列最重要的配件，S-Pen 的功能也有所提升，不但保留 Galaxy Note 7 S-Pen 具备 0.7mm 极细笔尖、4096 阶感压特性，还新增其他功能，包括「手写动态贴图」。用户可以在手机屏幕上写字或绘画，系统将把书写过程制成 GIF 档案，用户还可以即时发送相关档案。另外，S-Pen 也添加即时翻译功能，让人惊艳之处便是 S-Pen 只要放在外语文字上，系统就会即时翻译整行句子，而非像以往那样只是翻译单字！

Galaxy Note 8 还新增全新的「App Pair」功能，用户可以凭借一键同时启动两个应用程式，把“一心多用”发挥得淋漓尽致！

硬体规格方面，Galaxy Note 8 与 Galaxy S8 一样，分有 Snapdragon 835 处理器与 Exynos 8895 处理器两个版本，内存统一为 6GB RAM！存储空间分为 64GB、128GB 与 256GB，蓝牙 5.0，支持 NFC 与 MST 技术、与 Galaxy S8 系列一样支援虹膜、指纹及脸部辨识功能、支援 Bixby 人工智能助理、支援 Samsung Pay、电池容量为 3300mAh。另外，不管是手机本身或是 S Pen 都可以支援 IP68 防水。

三星 Galaxy Note 8 提供包括黑、蓝、灰与金四款颜色，然而官方并没有在发布会上宣布手机价格。不过，英国三星官网页面上已经标示了 869 欧元（约 4392 令吉），而美国营运商也确认，Galaxy Note 8上市日期为9月15日，从24日开始接受预购，一般售价为 930 美元（约 3700 令吉）。消息指出，若在预购期间下单，还可获得价值 230 美元 （约 1162 令吉）的三星 Gear 360 相机或是三星 128GB EVO +存储卡、快速无线充电底座。赠品相当吸引！至于马来西亚什么时候开始接受预购？售价是否如海外那般高？赠品又是什么呢？一切有待大马三星的官方宣布咯！
