OPPO首部全面屏手机！2日登陆大马

F5，对很多使用电脑上网的人来说，是刷新网页 (Refresh) 的意思。

OPPO F5作为OPPO最新推出的全面屏手机，一改了此前R9s、F1s的设计。为了全面屏，OPPO F5因此取消了 Home 按键，改用虚拟键，并首次将位于屏幕下方的指纹识别器移去手机背面。

OPPO F5显示屏高达6寸，采用了18:9的比例屏幕，分辨率为FHD+ (2160 x 1080)，因此机身上下部位也减窄了，而机身四角都采取了圆角过渡的设计，整机看起来十分顺滑，握持感也很舒适。

OPPO F5继承了OPPO F3主打摄像头的概念。在机身正面上方，F5搭载了高达2000万像素（f/2.0，背景虚化人像模式）的前置摄像头，搭载AI Beauty技术的重点加持，让自拍更完美。

后置镜头也不遑多让。机身背面嵌入了高达1600像素的镜头。前后镜头都支持LED闪光，让人最大限度的拍出满意的效果。
OPPO的标记位于背面中间，长圆形的指纹识别器位于它的上方。

很多手机用户都会使用手机听音乐，而是否拥有耳机插孔是众多用户选择手机时的考量之一。OPPO F5于机身下方保留了耳机插孔，分别和扬声器位于USB插电孔左右。

OPPO早前已在菲律宾发布了全新 F5，其规格配置如下：MediaTek mt6763T 8x 2.50GHz A53 处理器，4GB RAM /6GB RAM + 32GB/ 64 GB ROM（支援扩充），3200mAh电池容量，价格为15,990PHP（约RM1308）。

至于马来西亚的规格配置与价钱到底为多少，OPPO即将在11月2日正式推介！粉丝们就耐心等待吧！
