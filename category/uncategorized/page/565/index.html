有了红米还不够 小米将推全新品牌”蓝米”?

有不少消息流传，指小米即将推出全新的子品牌，不过消息一直未获得证实。最近，专门向国际介绍中国智能手机的英文网站 GizmoChina 就报道，小米即将推出一个名为“蓝米”的新品牌，与劲敌 OPPO 和 vivo 抗衡，一较高低！


图片摘自网络

报道指出，蓝米将推出的首部手机便是先前曝光的中端机 — X1。传言指蓝米 X1 有两种规格配置可选，一款将搭载全面屏、后置指纹；第二款非全面屏、前置指纹，至于其他配置别无二致，将采用 5.5 英寸显示屏 (消费者可选择 2K 或 1080p 解析度两种规格)、Snapdragon 660 处理器、4GB / 6GB RAM + 64GB / 128GB ROM，其中一款还会采用双摄像头设计。

图片摘自网络

早前有消息指出，小米有一款全新系列的手机 X1 将随 MIUI 9 一同发布，综合消息来看，这款新机将是蓝米的首发产品。虽然小米已经有了红米，但再创第三子品牌蓝米也不是不可能的。分析认为，小米这几年在中国市场的份额已被其他中国品牌取代，迫使小米必须思考不一样的行销策略，为此决定推出”蓝米”，小米主攻高端市场，红米顾及低端用户，而蓝米则主打中端产品。至于蓝米何时面向市场并发布 X1 手机，则有待官方的正式宣布咯！
