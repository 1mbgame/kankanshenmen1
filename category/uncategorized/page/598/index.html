太神了！薄荷糖果盒变身“想象不到的创意”

薄荷糖吃完了，小小的铁盒留着不知怎么处理，弃之又可惜，怎么办呢？外国一名“改造大神” Sudomod 就想到别人想象不到的创意，将糖果铁盒变成了游戏机！


这么小的游戏机？是的，随身携带多方便，这部糖果盒游戏机，是 Sudomod 自制的第二代「 mintyPi」。有别于第一代「 mintyPi」，这次他仍以 Raspberry Pi 系统作为效能来源，安装系统底板后，再以3D 打印外壳固定掣位和小屏幕就完成「mintyPi 2.0」。这部游戏机可谓麻雀虽小，五脏俱全，是一部集50款经典游戏于一身的迷你游戏机！「mintyPi 2.0」不但支持 USB 音效卡和 Wi-Fi，还支援 Game Boy、NDS、N64、红白机、PlayStation、PSP 等游戏。

「 mintyPi」 配置一个显示屏、简单的控制器和数个按键，还内置支架固定屏幕位置。和第一代比较，第二代的屏幕尺寸更大、音质更佳，还加入 Wi-Fi 功能，可以无线更新系统和游戏。实在是不能不佩服改造大神的创意！
