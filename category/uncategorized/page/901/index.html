不止地图那么简单！Google Map 助你避购物人潮

谷歌地图 Google Map 在很多人的日常生活中，扮演极其重要的角色。迷路或是出国旅游时，不用再开口问路，可以直接问谷歌。谷歌地图不但会提供公共交通选项，还能预估抵达目的地的时间。但显然的，谷歌不希望 Google Map 只是一般的地图。

在黑色星期五和感恩节购物热潮来临之际，谷歌宣布 Google Map 和 Google Search 的数项新功能，其中一项便是让你知道哪里人潮拥挤，帮助你避开人潮！



Google Map 通过谷歌搜索某个特定广场、商家，让用户知道那里是否挤满购物人潮，还会预估有关店铺、餐厅、超市等等地方什么时候较为拥挤、当前有多拥堵、顾客们在某个场所逗留的典型时间等等相关信息。这项新功能为用户带来便利，如果用户在出门前发现想去的商场挤满人潮，可以改变计划或是改为其他时段再去，既方便安排行程，又能在不拥挤，不需排长龙的情况下轻松购物。

其实，谷歌是透过谷歌上储存的用户信息，收集用户同一位置的数据，推算某一地点或是商店当时的用户人数。谷歌地图还会提供各种信息，包括药房什么时候出现顾客人流，超市的营业时间等等。在圣诞节和新年来临之际，或许这项功能也能帮助你躲过人潮，轻松购物！
