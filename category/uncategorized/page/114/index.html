小米6放大版报到！小米Note3旗舰规格中阶价

小米在11日下午举行新品发布会，正式推出小米MIX 2 【小米MIX2正式发布！号称最完美全面屏“仿佛握着一块玻璃”】。除了小米MIX 2 之外，还有另一款旗舰机同场加映，说的就是号称小米6的放大版 — 小米Note 3！


如果你喜欢小米6的外观设计，同时喜欢大屏幕，并在乎拍照与自拍效果，那么小米Note 3必定能满足你的需求。小米Note 3之所以被称为“小米6放大版”，是因为其外形设计与小米6基本相同，机身采用四曲面玻璃以及金属中框设计，不过边框换成7系铝边框，机身重量为此比小米6 轻了6g。

作为主打美颜自拍的手机，小米Note 3 配有1600万前置镜头，继承小米6的后置双摄方案，后置1200万像素双镜头，采用索尼IMX 386传感器，支持四轴光学防抖。另外，小米Note3也提升了相机人工智能算法，使到边缘识别更精准，背景虚化更自然，不管男女都能拍出效果更好的照片。


值得一提的是，小米Note 3还支持面部识别解锁功能，精度高、速度快。不管是湿手或是戴着手套，只要对上一眼就能轻松解锁。

小米Note 3参数规格
  • 5.5寸1080p 屏幕（护眼屏）
  • Snapdragon 660 八核处理器
  • 6GB RAM + 64GB/128 GB ROM 存储组合
  • 前置1600万像素AI美颜自拍
  • 后置1200万像素双镜头（2倍变焦、四轴光学防抖）
  • 3500mAh 电池容量
  • 人脸解锁、无孔式指纹识别
  • 支持快充3.0
  • 支持多功能NFC

小米Note3 提供亮黑与亮蓝两种机身配色，三种版本的售价与小米6一样。
  • 6GB RAM + 64GB ROM （亮黑版）   — 2499人民币（约1609 令吉)
  • 6GB RAM + 128GB ROM （亮黑版） — 2899人民币（约1863 令吉)
  • 6GB RAM + 128GB ROM （亮蓝版） — 2999人民币（约1927 令吉)
