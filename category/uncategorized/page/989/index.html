Google Pixel 防水级别低？ "输三星、苹果九条街"！

市面上各大品牌的旗舰智能手机标新立异，硬体规格也越来越强大。如今，防水功能已然成为智能手机必备或流行的规格之一。三星 Galaxy Note 7、S7、S7 Edge 、索尼 Xperia 系列都备有防水功能，而最近发布的苹果 iPhone 7、iPhone 7 Plus 也顺应时代需求，终于加入防水行列。不过，根据外国媒体 Android Police 爆料，价格应该不亚于 iPhone 7 的 Google Pixel XL，虽然也赶搭“防水列车”，但防尘防水级别却远远不如其他旗舰手机，只有 IP53。


图片: http://www.androidpolice.com

苹果 iPhone 7 的防尘防水级别为 IP67，三星和索尼则为 IP68，这表示，手机可在最多或超过 1 米水深下正常运作 30分钟。那么，谷歌 Pixel 的 IP53 到底有多大差别呢？

防尘防水 (IP) 等级由两个数字组成，Google Pixel 符合 IP53。5代表防尘功能，等于可以抵挡大部分灰尘进入配备，但因未达到最高等级“6”，所以不完全防尘。

IP53的”3″ 代表防水功能 (最高级别为8)，即是在低水压或垂直60度向机身喷水，手机在5分钟内还是可以正常操作，但若掉入水里，势必“凶多吉少”！因此，相较于苹果和三星，Google Pixel 作为谷歌年度安卓旗舰设备，防尘防水功能已经“输了九条街”。

图片: http://mobile.pconline.com.cn

根据日前曝光的消息，由 HTC 替 Google 代工的 Pixel 和 Pixel XL 将搭载骁龙 S820 或 S821 处理器、采用4GB RAM ，配有1200 万像素的摄像头。那到底输在“低防水级别”的 Pixel 系列，还有什么未“晒冷”的强大配备？ 10月4日，自有分晓。
