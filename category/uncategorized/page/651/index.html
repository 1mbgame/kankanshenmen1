变种病毒每秒"毒"一台电脑 3.0版本向全球扩散！

尽管 WannaCry 勒索病毒的扩散速度已经放慢，中毒的电脑数量也没有再增加，但根据英国《每日快报》报道，勒索病毒 WannaCry 已经出现新变种，而让人震惊的是，新变种的病毒每小时能感染3600台电脑，也就相等于每一秒就有一台电脑会中毒！

图片: http://www.express.co.uk

网络安全公司Check Point Software Technologies 强调，为了阻止病毒持续扩散，该公司已经采取紧急措施，启动病毒中的“自动销毁机制”。尽管如此，至今已有超过4万台电脑感染变种病毒。WannaCry 勒索病毒从上周五爆发一来，全球至少150个国家的30万台电脑“遭遇不测”。
欧洲刑警组织发言人指出，“目前，受病毒感染的电脑数量看似没有再增加，我们最终会得到一种解密工具，但现阶段，病毒仍是摆在眼前的威胁，我们仍在灾难后的恢复模式中。”
随着专家两次“捣毁”勒索病毒的源头后，是否意味着“全球网络灾难”就此落幕？综合外国媒体报道，专家在受感染的电脑中发现，3.0 新变种的病毒已经悄然来袭，向全球扩散！专家认为，前两次出现的勒索病毒皆设置“自动销毁机制”(Kill Switch)，而且只有部分系统可以运作，为此相信有关病毒是黑客的失败之作，虽然杀伤力不大，但传播速度快。

图片: https://unwire.hk

然而，WannaCry 3.0 已经更改了 Kill Switch 网址，试图躲过防毒软件的侦察。不过，经过专家逆向破解后，随即采用同样的方法阻止病毒扩散，也就是赶紧为该网域注册。一旦注册相关网域，也就等同于启动“自动销毁机制”。所以按照目前的情况看来，变种病毒都被专家“搞定”了，希望黑客就此收手，全球网络安全危机得以落幕！
