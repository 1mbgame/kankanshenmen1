Android与Chrome OS终要融合？ Nexus 抢先试、新笔电otw！

Android 和 Chrome OS 融合的传言扰扰攘攘了7年，这次真的“分久必合”了吗？

外国知名科技网站 9to5Google  和 Android Police 纷纷报道，指谷歌将在10月4日推出安卓与 Chrome OS 混合新系统 — Andromeda (仙女坐)。重点是，谷歌正在 Nexus 9 平板上测试 Andromeda 的可行性，如果一切顺利，Andromeda 将率先出现在由华为代工的 Nexus 平板和“便携式笔记型电脑”上 ！这也表示说，谷歌正在开发一款搭载新系统的笔记型电脑！


图片: https://9to5google.com/

图片: https://techcrunch.com

根据两个科技网站的报道，新款笔记型电脑厚度约10毫米、是功能完整的笔记型电脑。软件方面将不逊于Windows和macOS等竞争对手，搭载 12.3 英寸显示屏、可转换为平板电脑模式。内部硬件则采用英特尔 i3 或 i5 处理器、8GB 或 16GB RAM、两个USB-C接口、背光的键盘，以及支持压力传感的触控板。

消息指出，这款产品的内部代号为“Bison” 或 Pixel 3，预料谷歌将在10月4日举行的发布会上，宣布统一安卓和 Chrome OS 系统，但笔记型电脑的发布时间可能定在明年第三季度。

其实，谷歌开发 Andromeda 系统已有一段时间，期间各种消息甚嚣尘上，但谷歌并没有推出有关两大系统融合后的实际产品。这次，我们会否在谷歌下周举行的发布会上，听到有关“Andromeda”、“Bison” 或是新款笔记型电脑的最新消息？大家只是拭目以待了。
