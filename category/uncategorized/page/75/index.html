官方预告:Honor新机即将来马！售价料低于RM700

荣耀官方在面子书专页发布一张图片，预告一款Honor新机即将来马！而这款新机无疑就是Honor 6A啦！Honor 6A采用金属喷砂工艺打造，颜值不容置疑，而且顺利通过严峻考验 — 200次滚筒跌落测试，力证出色品质。


图片摘自网络

Honor 6A规格参数
  • 5英寸720p显示屏
  • Snapdragon 430处理器
  • 2GB RAM + 16GB ROM/3GB RAM + 32GB ROM（可扩展至128GB）
  • 前置500万像素镜头（十级美肤、智能补光）
  • 后置1300万像素镜头（0.5秒急速对焦）
  • 电池容量3020mAh（支持智电5.0省电优化技术）
  • 支持指纹识别
  • 支持双卡双待
  • 运行Android 7.0 的EMUI 5.1 系统

Honor 6A拥有金色、银色、玫瑰金和蓝色四种机身配色，暂时不清楚什么配色将引进我国。Honor 6A 2GB RAM + 16GB ROM在中国售价为799元(约510令吉)，而3GB RAM + 32GB ROM版本则售价999元(约638令吉)，预计本地售价将低于700令吉！
