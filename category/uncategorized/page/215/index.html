比苹果三星更快！这家中国厂商将首发屏下指纹手机

随着全面屏手机越来越普及，实现屏下指纹就是各厂商锁定的下一个目标，不过因为难以突破的技术障碍，苹果iPhone X与三星Galaxy Note 8并没有为市场带来惊喜。不过，现在几乎可以肯定，vivo将会发布全球首款屏下指纹手机！

图片：《福布斯》

指纹识别模块厂商Synaptics(新思国际)日前宣布推出全球首款屏下指纹传感器 – Clear ID FS9500，并透露与全球五大厂商之一合作，开始量产FS9500。

外国媒体《福布斯》爆料，这家与Synaptics合作的厂商就是vivo。消息曝光后，Synaptics官方推特也转发这篇报道，所以vivo将首发配置屏下指纹识别传感器几乎是板上钉钉的事了！根据了解，Synaptics将在明年一月举行的CES消费电子展上，展示首款屏下指纹智能手机。


从照片看来，这款vivo新机屏幕下方有一个指纹模样的形状，相信屏下指纹识别技术是通过触摸对应区域解锁。据称，指纹识别传感器可以直接封装在屏幕下方，最多占用1.5mm的厚度，采用光学技术，具有抗油污、防水、灰尘等特点。虽然照片中的vivo新机处于工程机状态，但上下边框看起来极窄，相信屏占比将进一步提升，视觉效果更好。

gif摘自网络

vivo曾在今年6月举行的MWC大会上，率先展示搭载屏下指纹识别功能的原型机，主要采用高通下一代超声波指纹解决方案。
