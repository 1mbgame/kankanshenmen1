原来不止iOS 苹果Mac OS X也存在安全漏洞！赶紧升级去吧！


图片取自technewstoday



上个星期，苹果公司才推出了iOS系统更新以修补遭到网络间谍公司NSO Group攻击的漏洞，此事一出，马上引起社会哗然，苹果系统不是向来号称不受骇客入侵，相当安全的吗？算了，反正苹果很快就让大家升级，及时修补了漏洞，大家也就安心了。


然而，事情并不是那么简单。同样的漏洞原来也出现在Mac OS X和Safari浏览器，而且不管是Yosemite或是El Capitan的版本都同时存在风险！根据Lookout Security 和 Citizenlab的说法，漏洞是透过传送恶意链接给人权运动人士，然后再藉此攻击他们的电脑，相信有关的方法就是由网络战争间谍NSO Group出售给阿联酋政府的。

原来不止Iphone iOS 苹果Mac OS X也存在安全漏洞！赶紧升级去吧！
不过你也不用太担心，苹果公司已经在9月1日释出了Mac OS X的安全升级版，不想成为攻击对象就赶快去更新吧！

身为苹果使用者你现在要做的是：

1. 升级你的iOS系统到9.3.5版本
2. 升级你的Mac OS X到 Yosemite 10.10.5 或 El Capitan 10.11.6版本
3. 升级你的Mac OS Safari到 9.1.3版本
