爱尔兰地标化身巨型竖琴 三星Galaxy Note 8广告“太好听”！

三星最新旗舰机Galaxy Note 8以“机皇”之姿攻占市场，三星不惜砸重金以“Do bigger things”为主题，为机皇宣传造势，刺激销量。爱尔兰三星近日发布一支2分钟13秒的最新宣传广告，夜光下的标志性大桥化身巨型竖琴，画面让人过目难忘，而竖琴演奏的“A Song for the City”(给城市的一首歌)更是悦耳动听，让人听得如痴如醉。

广告中这座大桥是爱尔兰著名地标塞缪尔贝克特桥（Samuel Beckett Bridge），三星通过特制软件将这座桥与竖琴演奏家艾斯林恩尼斯（Aisling Ennis）演奏的真实竖琴相连，使得大桥上的电缆与竖琴一一对应。当艾斯林恩尼斯拨动琴弦时，大桥的电缆也会随之发光。弹奏过程透过Galaxy Note 8 的屏幕演示，画面看起来美感十足。爱尔兰三星9月27日在面子书上载这个广告至今短短几天，播放次数已经超过23万。


塞缪尔贝克特桥出自Spaniard Santiago Calatrava的设计，他的创意灵感源自爱尔兰的传统竖琴，创意机构Guns or Knives 这次与投影制作专家Pauric Hackett合作，打造视觉与听觉的美学飨宴，为一代旗舰机打造非一般的宣传视频，务求达到更佳的营销效果。
