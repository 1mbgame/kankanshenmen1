Note 7 爆炸“船过水无痕”？三星竟能获此殊荣！

三星电子因为 Note 7 电池过热引发自燃事故而“疲于奔命”，最后基于安全考量，决定全球停产、停售一代旗舰手机，让三星蒙受前所未有的财务和名誉损失。然而，Note 7 事故似乎没有对韩国企业构成太大影响，三星电子仍能六度荣获至高殊荣，是韩国人心目中的“最佳企业”。


图片: www.youtube.com

根据《韩联社》报道，韩国品牌价值评估公司 Brandstock 近日发布的《2016年韩国百强品牌》评估报告显示，三星电子击退超过一千个品牌，名列品牌价值第一名，Galaxy 连续6年稳坐冠军宝座！尽管三星电子今年遭受 Galaxy Note 7 停产、停售事故而蒙受史无前例的损失，但仍无损品牌价值。三星 Galaxy 以924.2分摘下最具价值品牌冠军，满分为1000。 

图片: http://www.dailymobile.net/

韩国品牌价值评估公司 Brandstock 日前评估了韩国 230个领域，超过一千个品牌，三星电子“不负众望”稳居榜首，韩国超市品牌“emart” 以 908.7 分屈居第二，而韩国聊天应用程序 Kakao Talk 名列第三。
