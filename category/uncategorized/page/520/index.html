红米Note 5A曝光！这样的配置售价RM650？

小米手机今年来势汹汹，凭借广受好评的小米6、红米 Note 4X、红米 4X 和 5X 摆脱前两年的颓势，在市场份额上重新回到世界第五的位置。如此漂亮的翻身战，让人对小米接下来发布的新机充满期待与想象。最近，红米 Note 5A 悄然曝光！


爆料大神 @Slashleaks 曝光红米 Note 5A 的渲染图，乍看之下，疑似红米 Note 5A 的新机和上一代红米 Note 4X 有几分相似，但配置与参数分别甚大。从曝光的渲染图看来，红米 Note 5A 采用全金属机身设计，采用 5.5英寸 720P 显示屏，搭载骁龙425处理器， 2GB RAM + 16GB ROM，3080mAh 电池容量，前置 5000 万像素摄像头，后置 1300 万像素镜头，运行 Android 7.1 系统，而且还取消了指纹识别！

截图自网络

电商平台还曝光 Note 5A 的价格，预计售价为 999 元或约 649 令吉。（没有指纹识别、720p 显示屏，存储空间有限的手机，敢卖 649 令吉？） 另外，@Slashleaks 曝光的消息指红米 Note 5A 将搭载骁龙 425 处理器，和早前曝光的骁龙 625 处理器有所出入，到底谁是谁非？唯有等官方正式发布消息才能得知了。
