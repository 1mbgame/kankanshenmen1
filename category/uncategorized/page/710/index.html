折叠式手机比Note 8更早亮相？三星高管有话说！

三星电子一周前才发布新一代S系列手机 Galaxy S8，在这部旗舰机还未正式全球开卖之际，有关三星 “最黑科技” Galaxy X 和 Galaxy Note 8 的消息陆续有来。


图片摘自网络

一名对三星内幕熟知的 Galaxy 俱乐部成员在推特爆料，指三星下一代平板旗舰 Galaxy Note 8 将在今年第四季度亮相。但在 Galaxy Note 8 发布之前，还有一款备受瞩目的手机即将推出市场，说的就是备受业界关注的折叠式手机 Galaxy X。消息指出，Galaxy X 有望在今年第三季度面市。

今年？！那么快啊？Samsung Display 一名高管 Kim Tae-woong 接受《韩国先驱报》访问时回应此事，宣称今年不会推出折叠式手机，因为技术尚未成熟，这款手机有望在 2019年才推出市场。他说，无边框曲面屏手机销售形势良好，避免冲击曲面屏销量，三星不急于推出折叠式手机。这么一来，三星有充足的时间开发可折叠屏幕，毕竟 Samsung Display 还有一些技术挑战需要克服。

图片摘自网络

Samsung Display 是全球数一数二的移动屏幕制造商，该公司几乎垄断全球手机专用的OLED 屏幕，也是苹果十周年版手机 OLED 屏幕的独家供货商。Samsung Display 之前曾透露，可能先推出“单向”（上下或左右折叠）屏幕，之后再推出“双向”可折叠屏幕。

过去数个月来，三星向韩国知识产权局递交的专利申请一再曝光。三星在专利申请中展示一款狭窄的三星手机，手机的屏幕可以弯曲，还能够折叠成翻盖式手机，可以半自动化地折叠或展开。彭博社也曾报道，三星计划推出代号为 Project Valley 的屏幕可弯曲、可折叠的全新智能手机和平板电脑，展示三星在未来手机设计和体验方面，依然怀着雄心壮志。显然的，三星经历过 Galaxy Note 7 遭遇不测事故后，或许再度打起了可折叠式手机的主意，力求在市场上带来“革命性改变”。
