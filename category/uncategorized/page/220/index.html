Google Pixel不再是【最佳拍照手机】 记录维持半年就被这部新机取代！

每一部问世的旗舰机似乎逃不过知名独立相机测评网站 DxOMark 的测试，谷歌之前获得 89分，成为历来手机相机测评获得最高分的机型。不过，谷歌的记录维持大约半年，就被数天前刚刚发布的一款旗舰机给刷新了！


www.cnet.com

说的就是 HTC U11！HTC U11 经过 DxOMark 测试后，取得惊人的 90 分高分，把获得 89 分的谷歌 Pixel 给比下去！值得一提的是，Google Pixel 也是由 HTC 代工的！目前，排在 DxOMark 手机相机测评第二名的是 Google Pixel (89 分)，HTC 10、Galaxy S8 、S7 edge 和 Sony Xperia X Performance 同样以 88 分并列第三，HTC 可谓“大赢家”。

DxOMark 给 HTC U11 的好评包括在各种拍摄环境下都能拍出的稳定的成像品质，照片的细节细腻、杂讯噪点少、颜色准确、OIS 三轴光学防手震相当有效。HTC U11采用的 Dual Pixel PDAF 技术促进更好的相位对焦表现，就算在低光的环境下拍摄，照片品质依然受到肯定。总的来说，HTC U11 拥有几乎无可挑剔的影像功能，对焦、稳定、低噪点的部份获得高度评价。

DxOMark 也列出 HTC U11 相机的部分缺点，比如在户外拍摄时，照片会出现偏蓝的情况，远距离拍摄时，部分影像不够细致，还会出现亮度杂讯和残影等情况。

HTC U11 前置 1600 万自拍镜头（f/ 2.0 光圈），并有 Ultrapixel 模式，提升低光源时的自拍效果；后置 1200 万像素的 UltraPixel 3 镜头（1.4μm、f/1.7大光圈、光学防抖、Ultra-Speed 自动对焦、相位对焦）。

图片: http://phandroid.com

HTC U11 搭载5.5寸 2K 显示屏，配置高通最新的 S835 高阶处理器、4GB RAM/6GB RAM + 64GB ROM/128GB ROM 存储组合，运行 Android 7.1 Nougat 系统，支援 4G+3G 双卡双待，，具备 IP57 防水防尘级别，支援 QC 3.0 快充技术，3000mAh 电池，同时取消了 3.5mm 耳机接口。

HTC U11 主打的功能是 Edge Sense，主要让用户透过按压边框的方式，执行特殊功能，包括快速开启相机，再次按压边框就可以拍照。Edge Sense 可以让用户自己定义设置，可以开启应用程序、语音助理、手电筒等等，也可以在不同的应用中设置不同的功能，比如长按手机边框后，后置镜头会切换至自拍模式等等。
