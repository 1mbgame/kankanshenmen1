手机退场但功能“完美再现”！Note 7 部分功能转移至“两款”手机

尽管三星 Galaxy Note 7  出师不利，甚至走到了全球回收、全面停产的地步，但这并不代表一代旗舰手机的独特功能或软件就此消息！是的，Galaxy Note 7 的部分功能将被保留下来，并转移到 Galaxy S7 和 S7edge！


图片: http://www.androidauthority.com

如果你拥有 Galaxy S7 或 S7edge，只要通过最近一次的升级，就能获得 Galaxy Note 7 的部分功能，包括备受用户好评的“常亮显示” (always-on-display) 功能。三星数天前推送了最新升级版系统，用户不但可以定制屏幕，让屏幕呈现一张照片或时钟图案，并自行设置字样或颜色，还可以显示通知信息，正在播放的音乐等等。不过，需要 S Pen 支持的元素并不能在没有 S Pen 的 S7 和 S7edge 上正常显示。三星强调，always-on-display 并不如用户想象般耗电，每个小时消耗不到1%的电量。

图片: http://www.androidauthority.com


分析认为，三星这么做是希望让部分从 Galaxy Note 7 转移到 S7 和 S7edge 的用户，用新手机时较快上手，并淡化手机被降级的不良感觉，同时增加 S7 和 S7edge 的功能价值。所以说，既然 Galaxy Note 7 非退不可，那么 S7  和 S7edge  或许就是最好的选择。虽然 S7 和 S7edge 没有 S Pen 带来的笔记功能，然而在部分功能上，Note 7 和 S7 还是最为相似的两款手机。

大马三星电子日前发布文告，强调大马的 Note 7 用户可以要求退款或是更换成 Galaxy S7edge，并敦促 Galaxy Note 7 用户马上关机，以策安全。
