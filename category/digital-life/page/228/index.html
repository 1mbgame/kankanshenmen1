梁朝伟代言的小米Note 2 来了！“双曲面”设计最特别！

中国知名手机品牌小米暌违一年九个月后，终于在 25日发布最新机型小米 Note 2，还邀请“电眼影帝”梁朝伟代言，现身发布会畅谈最新手机！小米 Note 2 打着“一面科技，一面艺术” 的口号登陆高端机市场，梁朝伟形容自己是 “一面一代宗师，一面花样年华”，引起在场粉丝尖叫连连，也让人们对“一面科技、一面艺术”印象深刻。


图片摘自网络


小米 Norte 2 最大卖点是“双曲面”！该机采用金属中框加双面玻璃的设计，无论是正面或是背面都采用 3D 玻璃，让手机颜值升至另一个高度，传承小米 Note 的高颜值特色。手机的正面配备 5.7 英寸 OLED 材质柔性屏，拥有 77.2% 超高屏占比、110% NTSC 高色彩饱和度以及 100000：1 超高对比度，边缘曲面设计不但具有透明感、光泽感，还增添一丝时尚感。



硬体配置方面，小米 Note 2 搭载高通骁龙 821 处理器、主频达到 2.35GHz、4GB RAM +64GB ROM / 6GB RAM + 128GB ROM /  6GB RAM + 256GB ROM、具备 HD 高清音质、GPS 高精度定位、4070mAh 大容量电池。

摄像头方面，小米 Note 2 搭载 IMX318 超⾼像素相机，达到 2256万像素、最大光圈 F2.0 、4K 视频防抖，前置摄像头 800万像素，还有自动对焦功能。

图片: http://mt.sohu.com


小米 Note 2 拥有两种配色，分别是亮黑和冰川银， 4GB RAM +64GB ROM 版本售价为 2799 元（约1750令吉）、另外两种版本则分别售价 3299 元（约2000令吉）和 3499元 (约2144令吉)。小米 Note 2 将从11月1日起，在小米商城和小米之家开售。
