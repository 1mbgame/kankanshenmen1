HP推出OMEN系列电脑，售价从RM4499起！

HP推出最新的游戏笔电系列OMEN Desktop、OMEN 15及OMEN 17，售价从RM4499起！

OMEN Desktop采用战斗铠甲为设计主题，整体感觉时尚又充满战斗力，尤其注重在散热方面的性能，同时在主机上也配备了两个可热插拔的硬盘驱动器插槽。



OMEN 15及OMEN 17同样采用红黑设计，外形是一个超级酷炫的X字型，键盘方面也配合机身设计采用全红色，甚至在W、A、S、D特别采用了不同触感的键帽，让整体看起来更有型。

【OMEN Desktop】

-第7代Intel Core i5或i7
-重量：17.6kg
-体积：192mm x 442.1mm x 417.2mm
-可同时安装NVIDIA 1080 Ti显卡或双Radeon RX 580显卡
-Intel Kabylake Z170 或 H270主板
-16GB DDR RAM
-2TB 7200rpm HDD
-售价：RM5999


【OMEN 15】

-Windows 10 Home系统
-Intel Core i7-7700HQ处理器
-重量：2.57kg
-体积：388.5mm x 275.5mm x 24.8mm
-可选Radeon RX 550或GeForce GTX 1060显卡
-DDR4 RAM（最高支援16GB）
-15.6寸 FHD 120Hz屏幕
-1TB 7200rpm HDD
-256GB M.2 SSD
-售价：RM4499


【OMEN 17】

-Windows 10 Home系统
-Intel Core i7-7700HQ处理器
-重量：3.73kg
-体积：423mm x 304mm x 33mm
-可选GTX 1070或者RX 580显卡
-DDR4 RAM（最高支援32GB）
-17.3寸 FHD 120Hz屏幕
-1TB 7200rpm HDD
-256GB M.2 SSD
-售价：RM8899
