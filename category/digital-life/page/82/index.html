红米Note 5A正式发布 最低售价RM449！

小米21日晚间举行线上新品发布会，正式推出红米 Note 5A！值得一提的是，配置 5.5 英寸显示屏的红米 Note 5A 拥有较轻便的体型，重量才150克。

外观设计上，红米 Note 5A 与 OPPO、vivo 手机有点相似，但号称外形比 OPPO R9 与 vivo X7 颜值更高。红米 Note 5A 正面按照惯例采用对称美学设计，背面则采用金属机身及对称丝带元素设计，提供香槟金、玫瑰金和铂银灰三种配色，任君选择。



红米 Note 5A 配置规格（标准版）
  • 5.5 英寸 720p 显示屏
  • Snapdragon 425 四核处理器
  • 2GB RAM + 16GB ROM
  • 后置 1300 万像素摄像头 (支持 PDAF 相位对焦技术)
  • 前置 500 万像素镜头
  • 3080mAh 电池

与标准版不同的是，红米 Note 5A 高配版采用更高规格的处理器，并配置升级版内存组合，机身背面还增加指纹识别模块。相机方面则升级前置摄像头，辅以柔光闪光灯，呈现更好的自拍效果。

红米 Note 5A 配置规格（高配版）
  • 5.5 英寸 720p 显示屏
  • Snapdragon 435 八核处理器
  • 3GB RAM + 32GB ROM/ 4GB RAM + 64GB ROM
  • 后置 1300 万像素摄像头 (支持 PDAF 相位对焦技术)
  • 前置 1600 万像素摄像头（柔光闪光灯）
  • 后置指纹识别
  • 3080mAh 电池

红米 Note 5A 的另一个卖点是，手机采用 2+1 卡槽设计，也就是说两个 SIM 卡与一个 MicroSD 卡可以同时使用（容量最高可扩展至128GB）。售价方面，红米 Note 5A 标准版要价 699 元（约 449 令吉）；3GB + 32GB 版本售价 899 元（约 577 令吉）；4GB + 64GB 版售价 1199 元 （约 770 令吉）。好消息是，红米 Note 5A 将在八月杪升级到最新的 MIUI 9系统，至于红米 Note 5A 什么时候引入我国，会不会引入我国？则有待大马小米官方宣布咯！
