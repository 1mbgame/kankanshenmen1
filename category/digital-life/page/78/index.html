三星发布“双摄中端机” 价格比Galaxy Note 8便宜逾半！

三星旗下首款搭载双摄像头的旗舰机皇 Galaxy Note 8 终于推出市场，并从今天开始接受预购。虽然明白“一分钱一分货”的道理，但高达3999令吉的售价还是让不少消费者却步。如果你想购买三星推出的双摄手机，Galaxy Note 8 不是唯一的选择，三星最新推出的手机比Galaxy Note 8便宜60%！


三星发布了首部主攻中端市场的双摄新机 — Galaxy J7+！作为一部双摄手机，Galaxy J7+的配置算是诚意十足，后置1300万 + 500万像素镜头（f/1.7光圈 + f/1.9光圈），前置1600万像素摄像头，配有LED闪光灯，但没有配备光学防抖功能。

外观方面，Galaxy J7+ 采用5.5英寸1080P Super Amoled屏幕，全金属机身设计。除了配备Multi window、支援always on display功能之外，还支援智能助理Bixby以及脸部识别等其他功能。

Samsung Galaxy J7+ 基本配置
  • 5.5英寸Super AMOLED高清显示屏
  • 主频为2.4GHz的八核处理器 (官方未显示，料Mediatek Helio P20 处理器)
  • 4G RAM + 32GB ROM（可透过microSD扩展至 256GB）
  • 后置13MP + 5MP双摄镜头 (f/1.7光圈 + f/1.9光圈)
  • 前置16MP镜头 (f/1.9光圈)
  • 前置指纹识别
  • 3000mAh 电池
  • 运行Android 7.1.1 系统

三星Galaxy J7+提供金色、黑色与粉红色三种机身配色，在我们的邻国泰国首发，目前已经正式开放预购，9月22日正式开卖。这款新机何时会登陆大马市场，目前还不得而知，不过预计不会等太久。Galaxy J7+在泰国的售价为12900泰铢，折合马币大约1660令吉。
