三星Galaxy Note FE接受预购！未来一周“有买有送”

三星一代旗舰机Galaxy Note 7去年黯然离场，一年之后“重生版”— Galaxy Note Fan Edition重返大马市场。翻新版Galaxy Note 7日前已经在韩国发售，马来西亚是Galaxy Note FE进驻的第一个海外市场。


大马三星确认，Galaxy Note FE即日起至本月24日接受预购。好消息是，预购的用户将额外获得一年的手机保修期，以及一次性屏幕保修服务。预购手机的用户必须到特定的三星体验店(Samsung Experience Stores)或授权经销商处领取手机，并在10月25日至31日启动mySamsung账号，才能享有预购优惠。

Note Fan Edition与Galaxy Note 7外形一致，与Note 7最大的差异就是电池，电池容量从原本的3500mAh降到3200mAh，而且通过严厉的8级安全检测，保证用户用得安心。另外，三星在手机背面下方用镭射雕刻了“Galaxy Note Fan Edition”的字样，让用户一眼就能辨认这是为粉丝而重生的Note 7。

另外，Galaxy Note FE也配置Note系列少不了的S Pen，这次笔尖缩小以模仿签字笔的感觉，压感也提升到4096级以提供更好的触感体验。最重要的是这款S Pen手写支持IP防水防尘，即便荧幕有水，S Pen还是可以使用自如。S Pen可用来画画、录屏、制作GIF动态图、即时翻译和快速切换App等等。

三星Galaxy Note Fan Edition参数规格：
  • 5.7英寸2K Super AMOLED双曲面屏幕
  • Exynos 8890处理器
  • 4GB RAM + 64GB ROM
  • 前置500万像素镜头 (f/1.7大光圈）
  • 后置1200万像素Dual Pixel镜头 (f/1.7光圈、光学防抖）
  • 支持虹膜识别
  • 前置指纹识别
  • 支持Samsung Pay
  • 支援无线充电、快充
  • IP68防水防尘（可在1.5米水下坚持30分钟）
  • Bixby Home
  • 3200mAh电池容量
Galaxy Note FE将在10月25日正式在本地开卖，售价为2599令吉，预购期间仅提供黑色一种配色。
